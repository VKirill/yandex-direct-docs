# Настройка в Мастере кампаний

### Добавьте организацию в Яндекс Бизнес

Чтобы ваша организация появилась в Картах, добавьте ее через интерфейс выбора организации в рекламной кампании или через личный кабинет [Яндекс Бизнеса](https://yandex.ru/sprav).

Чем больше информации — тем заметнее и привлекательнее будет ваша компания для пользователей и сможет чаще попадать на рекламные позиции. Укажите название, контактные данные, рабочие часы, добавьте логотип, обложку, фотографии, добавьте товары и услуги. Подробнее о заполнении карточки организации в инструкции на портале Яндекс Бизнеса [Как заполнить профиль компании](https://business.yandex.ru/instructions/organization/?utm_source=adv&utm_medium=article&utm_campaign=article-link&utm_content=business).

Если ваша организация уже добавлена, [найдите](https://yandex.ru/sprav/add) ее и подтвердите права владельца.

### Выбор организации

Укажите ссылку на сайт. Компания автоматически появится в блоке с опцией **Продвижение организации из Яндекс Бизнеса**. Убедитесь, что опция включена.

Если организация не нашлась, добавьте ее вручную. Нажмите **Добавить организацию**. Если в списке нет нужной компании, нажмите **Создать организацию** и заполните ее профиль.

Если вы планируете продвижение без сайта, аналогично добавьте организацию вручную.

Убедитесь, что в профиле организации указана ссылка на продвигаемую страницу. Тогда при клике на текст объявления на карточке организации пользователь перейдет на продвигаемую страницу. Если в профиле нужной ссылки нет, текст объявления на карточке организации не будет кликабельным.

### Места показа

Если опция **Продвижение организации из Яндекс Бизнеса** включена, места показа **Яндекс Карты** и **Список организаций в результатах поиска** добавятся автоматически.

### Целевые действия

Выберите цель продвижения и целевые действия. Для более эффективного продвижения добавьте цели с сайта и цели счетчика для профиля организации. Этот счетчик создается автоматически для карточки организации и фиксирует такие действия пользователя в карточке, как клик по кнопке «Позвонить», клик по кнопке «Построить маршрут», переход в мессенджер и другие.

Примечание

При выборе цели продвижения «Максимум целевых действий» укажите хотя бы одну цель с сайта и одну цель счетчика для профиля организации. Без этого сохранить кампанию не получится.

### Настройка объявления

1. Укажите заголовок и текст объявления — они показываются в карточке организации в Картах и списке организаций в результатах поиска. Подробнее о [правилах составления](ru/efficiency/text-and-title) заголовка и текста.
2. Укажите [номер телефона](ru/efficiency/phone) организации. Этот номер будет отображаться в карточке организации в показах.
3. По умолчанию в объявление будет добавлена кнопка действия. Текст на кнопке и ссылка сформируются из целей счетчика Яндекс Бизнеса, настроенных в кампании. Кнопка будет отображаться в объявлении, если оно выиграло аукцион.

Примечание

Остальные элементы объявления в карточке организации не отображаются, например: изображения, видео, быстрые ссылки.

### Особенности показов

В Картах и в списке организаций в результатах поиска не будет показов:

- по [ключевым фразам](ru/keywords/keywords);
- по [интересам и привычкам](ru/products-text-image-ads/user-profile).

**География показов**

[Регионы](ru/efficiency/geotargeting), выбранные в настройках кампании, не влияют на показы в Картах или в списке организаций в результатах поиска. Например, пользователь из Тюмени ищет организацию в Твери. Ему могут быть показаны подходящие карточки организаций в выбранном регионе. При масштабировании карты могут показываться другие организации.

**Автотаргетинг**

Для эффективного продвижения в Картах должны быть подключены все [категории запросов](ru/impression-criteria/autotargeting). Для этого в блоке **Аудитория** оставьте настройку по умолчанию — **Подобрать оптимальную**.

Объявление организации будет показано только при совпадении запроса с тематикой рубрик организации и при наличии соответствующих товаров и услуг в профиле организации.

Примечание

В отличие от глобального поиска Яндекса, в Картах пользователи чаще всего ищут информацию по широким запросам.

Показов по запросам с упоминанием брендов в Картах не будет.
