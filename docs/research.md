# Исследования эффективности медийной рекламы

Чтобы оценить работу медийных рекламных кампаний, их воздействие на поведение пользователей и восприятие бренда, одновременно с запуском кампании проведите исследования их эффективности:

- [Brand Lift](./brand-lift.md) — показывает изменение брендовых метрик в ходе рекламных кампаний;
- [Search Lift](./search-lift.md) — оценивает, как реклама увеличивает количество поисковых запросов, связанных с брендом;
- [Visit Lift](./visit-lift.md) — анализирует, насколько реклама стимулирует посещаемость сайта;
- [Target Lift](./target-lift.md) — оценивает, как растет число целевых действий на сайте после просмотра медийной рекламы.

### Виртуальный аукцион

В ходе исследований участвуют две аудитории рекламной кампании:

- тестовая группа — пользователи, которые видели рекламу;
- контрольная группа — пользователи, которым реклама намеренно не демонстрируется.

#### Как формируется контрольная группа

Для создания контрольной группы используется виртуальный аукцион. Как и для обычного аукциона, в нем отбираются пользователи, которые соответствуют настройкам конкретной рекламной кампании и которым в другой ситуации реклама показалась бы.

Объявление рекламодателя участвует в аукционе, выигрывает его, но показ при этом не происходит. А пользователь, для которого предназначался этот показ, попадает в контрольную группу во время исследований.

Для таких пользователей происходит виртуальный показ — потенциально возможный показ объявления, который не был совершен, так как пользователя добавили в контрольную группу. Рекламодатель не платит за такие показы.

Виртуальный аукцион позволяет наиболее объективно сравнивать тестовую и контрольную группы в исследованиях, где оцениваются итоги рекламной кампании. Составы тестовой и контрольной группы максимально схожи, их единственное различие в последнем этапе, на котором контрольной группе не показывается реклама. Контрольные группы исследований Brand Lift, Search Lift, Visit Lift, Target Lift состоят исключительно из участников виртуального аукциона.
