# Как запустить показы для автомобилей

- [Требования к объявлениям](ru/product-gallery/cars#restr)
- [Создание кампании](ru/product-gallery/cars#auto-create)
- [Статистика](ru/product-gallery/cars#statistic)

В товарную галерею могут попасть объявления о продаже новых и подержанных легковых и грузовых автомобилей, спецтехники, созданные из фида. Текстово-графические объявления в Единой перфоманс-кампании о продаже автомобилей не попадут в товарную галерею.

## Требования к объявлениям

- По ссылке открывается страница с одним автомобилем.
- Актуальны наличие автомобиля и его цена в объявлении.

- Ссылка ведет на сайт с информацией о продвигаемом автомобиле. Переадресация на другие сайты не допускается.

## Создание кампании

Нажмите **Добавить** → **Кампанию**.

- В Мастере кампаний выберите **Товарная кампания** и создайте [объявления для товаров](ru/campaign-master/product-campaign).
- В режиме эксперта выберите **Единая перфоманс-кампания** и создайте [товарные объявления](ru/unified-performance-campaign/create-product).

Объявления можно создать на основе [XML-фида Авто.ру](ru/feeds/requirements-other-categories).

## Статистика

Статистика по показам в товарной галерее доступна в [Мастере Отчетов](ru/statistics/report-wizard). Для анализа размещения постройте отчет по срезу **Вид размещения** → **Товарная галерея**.
