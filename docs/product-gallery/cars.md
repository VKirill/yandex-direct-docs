# Как запустить показы для автомобилей

- [Требования к объявлениям](../product-gallery/cars.md#restr)
- [Создание кампании](../product-gallery/cars.md#auto-create)
- [Статистика](../product-gallery/cars.md#statistic)

В товарную галерею могут попасть объявления о продаже новых и подержанных легковых и грузовых автомобилей, спецтехники, созданные из фида. Текстово-графические объявления в Единой перфоманс-кампании о продаже автомобилей не попадут в товарную галерею.

## Требования к объявлениям

- По ссылке открывается страница с одним автомобилем.
- Актуальны наличие автомобиля и его цена в объявлении.

- Ссылка ведет на сайт с информацией о продвигаемом автомобиле. Переадресация на другие сайты не допускается.

## Создание кампании

Нажмите **Добавить** → **Кампанию**.

- В Мастере кампаний выберите **Товарная кампания** и создайте [объявления для товаров](../campaign-master/product-campaign.md).
- В режиме эксперта выберите **Единая перфоманс-кампания** и создайте [товарные объявления](../unified-performance-campaign/create-product.md).

Объявления можно создать на основе [XML-фида Авто.ру](../feeds/requirements-other-categories.md).

## Статистика

Статистика по показам в товарной галерее доступна в [Мастере Отчетов](../statistics/report-wizard.md). Для анализа размещения постройте отчет по срезу **Вид размещения** → **Товарная галерея**.
