# Товарная галерея

- [Места показов товарной галереи](../product-gallery/about.md#places-of-impressions)
- [Как запустить показы](../product-gallery/about.md#how-to-start)

Товарная галерея — карусель с карточками товаров из разных магазинов или авто от разных продавцов, которая показывается в результатах поиска. Алгоритмы самостоятельно определяют оптимальное место показа галереи, чтобы увеличить вероятность конверсии. В результатах поиска может быть показана только одна товарная галерея.

Преимущества товарной галереи:

- Показывает предложения, цены, акции и скидки в привлекательном и заметном формате.
- Помогает пользователю определиться с выбором.
- Повышает конверсионность продвижения товаров и автомобилей.
- Подходит бизнесу любого масштаба, в том числе небольшим магазинам.

В товарной галерее могут быть показаны:

- текстово-графические объявления из Единой перфоманс-кампании о товарах — с изображением, ценой и ссылкой на товар;
- текстово-графические объявления из Единой перфоманс-кампании о каталогах — с изображением и ссылкой на каталог;
- товарные объявления и страницы каталога из Единой перфоманс-кампании;
- объявления для товаров и страниц каталога из Товарной кампании.

Страницы каталога в товарной галерее доступны только для товаров. Добавить в нее подборки автомобилей или предложений других тематик, в том числе услуг, не получится.

В товарную галерею может попасть более 5 карточек одного бизнеса, то есть в выдаче могут быть карточки только одного магазина. При этом один и тот же товар не повторяется — пользователю показываются разные товары, соответствующие его запросу (например, несколько моделей прямых диванов).

Также в Товарной галерее могут отображаться товары, похожие на основной запрос пользователя. Такие товары будут показаны с пометкой «Похожий товар». Чтобы ваши товары могли показаться в таком формате, [выберите категории](../impression-criteria/autotargeting.md#autoy_on) **Альтернативные запросы** и **Запросы с упоминанием брендов конкурентов** в настройках кампании в разделе **Автотаргетинг**.

Выбор объявлений для показа происходит на основе [VCG-аукциона](../technologies-and-services/vcg-auction.md#calc-vcg). Он оценивает комбинацию ставки, коэффициента качества и прогноза CTR для конкретного показа. В галерею попадают объявления с лучшим сочетанием показателей.

## Места показов товарной галереи

Для всех товаров, кроме автомобилей:

- Поисковая выдача.
- Результаты поиска в картинках.
- Выдача в [поиске по товарам](https://yandex.ru/support/products/index.html).
- В приложении Едадил в результатах поиска.

Показы автомобилей будут только в поисковой выдаче.
