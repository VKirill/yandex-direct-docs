# Почтовые уведомления

Вы можете получать уведомления о состоянии ваших кампаний по электронной почте.

Почтовые уведомления настраиваются в [параметрах кампании](../campaigns/campaign-settings.md).

Какую информацию можно получать по электронной почте:

- об остатке средств;
- об изменении объема трафика;
- об остановке по достижении дневного бюджета;
- о готовности XLS-отчетов.
