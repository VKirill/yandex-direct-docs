# Запрещение показов по IP-адресам

- [Почему запрещение показов может навредить продвижению](ru/campaigns/disable-ip#section_hpb_py1_jfb)
- [Подозрительные действия с определенного IP-адреса](ru/campaigns/disable-ip#section_xq5_ty1_jfb)

Экспериментируя с настройками кампании, вы можете ограничить показ объявлений определенным пользователям по *IP-адресу*. На странице редактирования кампании откройте дополнительные настройки.

Невозможно запретить IP-адреса из частных подсетей (например, 10.x.x.x, 192.168.x.x), так как они предназначены для внутренних сетей компаний и в интернете не используются.

Чтобы определить, с каких IP-адресов заходили пользователи, посмотрите логи сервера, на котором находится ваш сайт. Сделать это можно с помощью специальных программ. О них вы можете узнать у своего провайдера. Тех, кто пришел на ваш сайт из объявлений Директа, можно выделить с помощью дополнительного параметра вида `?from=direct`, добавленного к ссылке на сайт, например `www.в-космос.рф/index.html?from=direct`.

## Почему запрещение показов может навредить продвижению

При выдаче IP-адреса есть особенности:

- При модемном соединении IP-адрес выдается динамически и может измениться при следующем подключении того же пользователя к интернету.
- Компании и провайдеры часто организуют доступ в интернет через прокси-сервер, и тогда все компьютеры, выходящие в интернет через него, имеют один и тот же IP-адрес.

В обоих случаях за одним и тем же IP-адресом может находиться большое количество не связанных между собой пользователей. Поэтому запрещать показы по IP-адресам не рекомендуется, так как это может привести к снижению количества потенциальных клиентов.

## Подозрительные действия с определенного IP-адреса

Мы не рекомендуем запрещать показы по IP-адресу, даже если вы считаете, что с него осуществляется подозрительная активность. Директ отсеет недобросовестные клики и показы с помощью специальной [технологии защиты от фрода](ru/technologies-and-services/antifraud).
