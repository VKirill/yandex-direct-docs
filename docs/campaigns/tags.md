# Фильтрация групп объявлений с помощью меток

- [Создание и редактирование меток](../campaigns/tags.md#create-tags)
- [Использование меток](../campaigns/tags.md#common-tags)

Метки позволяют объединять группы объявлений внутри кампании для быстрого поиска и редактирования. Вы можете посмотреть сразу все группы объявлений, которым присвоена одна и та же метка, и применить к ним общие действия.

С помощью меток удобно объединять группы объявлений по товарным категориям, брендам, маркам, стоимости товара, а также фильтровать статистику в [Мастере отчетов](../statistics/report-wizard.md#filtr).

## Создание и редактирование меток

1. Выберите кампанию.
2. На вкладке **Группы** выделите нужные группы и нажмите **Действия** → **Метки** → **+Добавить метку**.
3. Укажите слова или фразы через запятую. Максимальная длина метки — 25 символов, включая любые знаки, кроме запятой. Можно создать до 200 меток на кампанию. Группе объявлений можно задать не более 30 меток.
4. Нажмите **Применить** — метки будут присвоены группе.

Вы можете переименовать, удалить существующие или создать новые метки. Для этого в окне настройки меток нажмите **Управление метками**.

Скрыть раздел **Метки** можно на странице **Настройки пользователя**, отключив опцию **Показывать метки**.

## Использование меток

В Директе можно использовать уже созданные метки для фильтрации групп. Для этого выделите кампанию и перейдите к ее группам. Нажмите **Метки** и выберите одну или несколько меток из списка.

Чтобы добавить или удалить метку, нажмите значок .

В [Мастере отчетов](../statistics/report-wizard.md#filtr) используйте метки как условия фильтрации для анализа кампании. Например, вы объединили группы объявлений c помощью меток по брендам. При составлении отчета добавьте условие фильтрации «Метка» с нужными значениями. В отчет будут включены данные только по выбранным брендам.
