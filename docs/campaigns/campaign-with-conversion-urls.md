Кампании с конверсионными страницами

# Кампании с конверсионными страницами

- [Как работает](ru/campaigns/campaign-with-conversion-urls#kak-rabotaet)
- [Частые вопросы](ru/campaigns/campaign-with-conversion-urls#chastye-voprosy)

Директ автоматически выявляет страницы вашего сайта, которые приносят конверсии, но еще не продвигаются. На основе этих данных система создает рекомендации для новых кампаний или групп объявлений, помогая привлечь дополнительный целевой трафик и увеличить продажи.

## Как работает

Система анализирует данные о конверсиях на сайте — через Яндекс Метрику с подключенным [модулем Ecommerce](https://yandex.ru/support/metrica/ru/ecommerce/about). Сначала определяет все страницы, которые приводят к покупкам или другим целевым действиям. Затем отбирает из этого списка те страницы, которые в продвижении не участвуют, но за последний месяц с них был переход, который привел к конверсии. Для таких страниц Директ рекомендует создать кампанию или группу объявлений.

Рекомендации обновляются раз в неделю.

## Частые вопросы

**Товара нет в наличии на этой странице**

Система проверяет наличие товара по данным Ecommerce. Если товар недоступен или имеет статус «Предзаказ», страница не будет рекомендована.

Если данные о наличии не передаются, система не сможет определить статус товара.

**Это товар с низким средним чеком, его реклама не окупится**

Рекомендации формируются на основе потенциального дохода. Система учитывает конверсии, стоимость заказов (если данные доступны через Ecommerce) и их вероятность. В рекомендации попадают страницы, которые с наибольшей вероятностью принесут доход с учетом затрат на рекламу.

**Этот адрес страницы для другого региона, который я не хочу рекламировать**

Система учитывает геотаргетинг существующих кампаний и регион пользователя, совершившего конверсию. Рекомендации формируются только для регионов, которые активны в вашей рекламной стратегии. При возможности используется основной адрес страницы.

**Акция на этой странице уже закончилась**

Система еженедельно проверяет актуальность страниц (например, даты акций и активность конверсий). Страницы с устаревшими предложениями не попадают в рекомендации.

**Это техническая, информационная страница или страница пагинации**

Рекомендации формируются только для коммерчески значимых страниц каталогов: товаров по категориям, подборок («Новинки», «Хиты», «Скидки»), тематических подборок и промо-лендингов с актуальными предложениями.

В рекомендации не попадают следующие страницы:

- технические страницы (ошибка 404 (Not Found), ошибка 500 (Internal Server Error), результаты поиска по сайту);
- информационные страницы (о компании, доставке, контактах, блог без коммерческого предложения);
- страницы пагинации (page=2, page=3 и т. д.).
