# Финансовые пирамиды, хайп-проекты

При показе объявлений в Директе учитываются запреты и ограничения местного законодательства на распространение рекламы и рекламная политика Яндекса.

Нельзя продвигать финансовые пирамиды (в том числе организации, которые находятся в [списке](https://www.cbr.ru/inside/warning-list/) компаний c признаками нелегальной деятельности на финансовом рынке) и хайп-проекты.
