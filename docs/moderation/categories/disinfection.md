# Услуги по дезинфекции, дезинсекции и дератизации

В Директе можно продвигать услуги по дезинфекции, дезинсекции и дератизации.

Ваши объявления должны соответствовать законодательству тех стран, в которых вы хотите показывать объявления.

На продвигаемом сайте должны быть указаны [контактные данные](../../moderation/ad-rules.md#kontakty).
