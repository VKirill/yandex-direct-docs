# Cтимулирующие мероприятия

В Директе можно продвигать стимулирующие мероприятия — акции, конкурсы, мероприятия рекламного характера.

Согласно закону «О рекламе» стимулирующее мероприятие — это конкурс, игра или иное подобное мероприятие, условием участия в котором является приобретение определенного товара.

Для размещения не требуются специальные документы.

Если стимулирующее мероприятие позиционируется как лотерея, проверка проходит по правилам раздела [Лотереи](../../moderation/categories/lottery-lotteries.md#ru).

Продвигаемое мероприятие не должно подразумевать риск. В частности, товар, который покупатель приобретает для участия в стимулирующем мероприятии, должен оставаться у него. В противном случае мероприятие считается азартной игрой и объявление не будет принято.

Для продвижения стимулирующих мероприятий необходимы документы, перечисленные в разделе [Лотереи](../../moderation/categories/lottery-lotteries.md#by).
