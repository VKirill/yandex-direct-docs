# Книги, видео, сайты о медицине и лечении

В Директе можно продвигать тренинги, семинары, форумы, информационные порталы, статьи, книги, видео и другие источники информации медицинской тематики.

Не принимаются объявления, которые:

- подразумевают нежелательность вакцинации;
- содержат пропаганду против вакцин.

Ваши объявления должны соответствовать законодательству тех стран, в которых вы хотите показывать объявления.

Россия

Беларусь

Казахстан

Узбекистан

Другие страны
