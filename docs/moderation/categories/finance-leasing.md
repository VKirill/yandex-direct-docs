# Лизинг

В Директе можно продвигать лизинг.

Ваши объявления должны соответствовать законодательству тех стран, в которых вы хотите показывать объявления.

В ваших объявлениях с показами на Россию должны соблюдаться все требования закона «О рекламе». Например, графические объявления в Единой перфоманс-кампании о лизинге должны содержать наименование юридического лица и сведения о его организационно-правовой форме. Эти данные должны свободно восприниматься пользователем одновременно с основным контентом объявления, без использования им каких-либо специальных средств или совершения дополнительных действий. Шрифт должен читаться при минимальном разрешении. Информация должна присутствовать на экране в течение времени, достаточного для прочтения всего текста, присутствующего в кадре, при средней скорости чтения.

В ваши текстовые объявления предупреждение с наименованием юридического лица будет добавлено автоматически. Текст предупреждения зависит от данных, указанных в документах.

Подробнее о [предупреждениях](../../moderation/age-alerts.md).

Для размещения не требуются специальные документы.
