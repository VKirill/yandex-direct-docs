# Государственные органы и их услуги

Продвижение госуслуг размещается в Директе с ограничениями:

- Можно продвигать только помощь в получении документов или иных услуг, оказываемых государственными органами. Например, помощь в получении регистрационного удостоверения. Такое объявление не должно гарантировать получение документа. Если в качестве услуги предлагается помощь с обращением в государственный орган, из рекламно-информационных материалов должно быть понятно, что услугу оказывает посредник.
- Не принимаются объявления о получении выписок и помощи в получении выписок из Единого государственного реестра недвижимости.

Для размещения не требуются специальные документы.
