# Сигареты, табак и аксессуары

Объявления о табачных изделиях и аксессуарах для табака размещаются с ограничениями. Это связано с требованиями законодательства и политикой Яндекса.

Можно продвигать:

- [Аксессуары для табака](../../moderation/categories/smoking-tobacco-ex.md)

Нельзя продвигать:

- [Табачные изделия](../../moderation/categories/smoking-tobacco.md)

Ваши объявления должны соответствовать законодательству тех стран, в которых вы хотите их показывать.

[Другие требования к сайту и оформлению объявления](../../moderation/ad-rules.md).
