# Медиация

К размещению принимаются объявления об урегулировании споров с помощью посредника.

В ваших объявлениях и на странице перехода должны соблюдаться требования законодательства. Например, объявления не должны содержать утверждение, что медиация имеет преимущества перед разрешением спора в суде.

Для размещения не требуются специальные документы.
