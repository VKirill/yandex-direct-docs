# Правила общения в чате

1. Операторы отвечают на вопросы только о Яндекс Директе.
2. Относитесь к собеседнику с уважением. В чате запрещены:

   - враждебный тон;
   - любые оскорбления, в том числе связанные с полом собеседника;
   - сообщения с сексуальным подтекстом;
   - нецензурные выражения;
   - спам;
   - угрозы.

   Если пользователь нарушает правила, Яндекс может заблокировать возможность задавать вопросы в чате.
3. Вы можете оставить [отзыв о консультации](../troubleshooting/servicequality.md). Ответ от службы контроля качества придет по электронной почте.
