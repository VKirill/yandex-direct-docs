## Специалисты по настройке Директа

Для настройки, ведения или аудита ваших кампаний в Директе вы можете выбрать специалиста по настройке Директа. Отобранные нами специалисты прошли проверку знаний, оценку качества уже настроенных ими кампаний и получили именной электронный сертификат.

Узнать об условиях сотрудничества можно напрямую у специалиста. Чтобы начать сотрудничество, требуется создать [логин на Яндексе](../quick-start/create-campaign.md) и настроить [аккаунт в Директе](../quick-start/create-campaign.md).

[Выбрать специалиста](https://yandex.ru/adv/contact/freelancers)

## Агентства

С Яндексом сотрудничает множество рекламных агентств по всему миру. Сертифицированные агентства имеют опыт проведения кампаний в Директе и помогут решить ваши задачи. В списке агентств, работающих в вашем городе, вы всегда можете выбрать подходящее.

[Выбрать агентство](https://yandex.ru/adv/contact/agency?from=help)
