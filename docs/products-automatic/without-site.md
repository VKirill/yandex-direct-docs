# Запуск кампании для бизнеса без сайта

- [Создание кампании](ru/products-automatic/without-site#launch-campaign)
- [Что еще можно настроить](ru/products-automatic/without-site#chto-eshe-mozhno-nastroit)
- [Статистика кампании](ru/products-automatic/without-site#statistics)

Если у вас нет сайта, для размещения объявлений с Простым стартом достаточно написать пару предложений о своем деле, а нейросеть Яндекса поможет с остальным — создаст [лендинг](ru/efficiency/landing) c информацией о продукте и контактами, поможет с настройками, текстами и картинками.

Результаты кампании можно будет отслеживать в блоке [статистики](ru/products-automatic/without-site#statistics).

## Создание кампании

Нажмите **Добавить** → **Кампанию**. На вкладке **Простой старт** нажмите **Перейти к созданию объявлений**. В блоке **У меня нет сайта** нажмите **Продолжить без сайта**.

**Шаг 1. Создайте и настройте лендинг**

1. Выберите источник информации для лендинга: по описанию бизнеса или на основе сайта.

   По описанию

   На основе сайта

   1. Выберите **ваш статус**: организация или частный специалист.
   2. В поле **Название бизнеса** укажите название кампании или вид деятельности, например «Вышивка на джинсовках на заказ».
   3. В поле **Описание бизнеса** расскажите, почему стоит выбрать именно ваше предложение. Укажите конкурентные преимущества: скидки, акции, условия доставки, гарантию и т. д.
   4. Нажмите **Продолжить**.

   1. Выберите **ваш статус**: организация или частный специалист.
   2. Укажите адрес страницы сайта.
   3. Нажмите **Продолжить**.
2. В блоке **Организация из Яндекс Бизнеса** привяжите свою организацию, тогда на лендинг можно будет добавить форму заявки, отзывы и блок с адресом и картой. Вы также можете продолжить без добавления организации.
3. Настройте кнопки контактов (телефон, WhatsApp или Telegram).

   Важно

   Без контактных данных лендинг не сохранится.
4. Нажмите **Создать лендинг**.

   Запустится генерация контента лендинга, это может занять до минуты. После завершения генерации откроется редактор контента лендинга.
5. [Настройте контент лендинга](ru/efficiency/landing#сreate-content).

**Шаг 2. Настройте кампанию**

1. Найдите вашу компанию на Картах и добавьте в объявление, чтобы привлекать клиентов из Яндекс Карт. Так Директ поймет, какую организацию продвигать, и возьмет из ее профиля полезную информацию.
2. Подберите нужные регионы для показа объявлений. Можно показывать объявление на уровне региона, области или города.

   > Например, для маленькой кофейни подойдет город, где про нее смогут узнать ее будущие посетители, а для онлайн-магазина одежды — все регионы, куда можно доставить товар.
3. Поясните, чем вы занимаетесь. Начните писать и выберите 2-3 вида деятельности.
4. Алгоритм Директа уже подобрал несколько изображений для создания объявлений: с сайта, из фотостока или среди сгенерированных нейросетью. Если нужно, замените их на другие. Можно добавить до 50 вариантов.
5. Выберите один из вариантов бюджета, рассчитанных системой, или укажите свой. Суммы указаны без НДС.

   Бюджеты рассчитаны по статистике трат у конкурентов. Чем больше бюджет, тем выше вероятность, что вы выиграете аукцион и получите больше трафика.

   Сумма бюджета рассчитана на 7 дней. Директ будет расходовать средства в пределах заданной суммы. Кампания будет идти, пока на счете есть деньги.
6. Нажмите **Запустить кампанию**.

Яндекс как оператор рекламных данных при создании кампании передает всю информацию по размещению в Единый реестр интернет-рекламы (ЕРИР). Подробнее о [маркировке рекламы](https://yandex.ru/adv/ad-labeling).

Если на счете есть средства в размере выбранного месячного бюджета кампании, кампания запустится автоматически.

Прогресс-бар покажет, в каком статусе находится кампания и дальнейшие шаги:

- **«Ожидает оплаты»** — недостаточно средств. Под прогресс-баром появится подсказка, что нужно [пополнить счет](ru/payments/payment-methods).
- **«На модерации»** — объявления отправлены на модерацию.

  Результаты проверки на соответствие [требованиям](ru/moderation/adv-rules) вы получите в письме на адрес, указанный при регистрации в Директе.
- **«Активна»** — кампания прошла модерацию, начались показы объявлений.

## Что еще можно настроить

Информации, которую вы указали, уже достаточно — показы начались. Чтобы сделать продвижение более результативным, можно добавить больше рекламно-информационных материалов.

На странице кампании перейдите на вкладку **Рекламно-информационные материалы**.

1. Заполните анкету.

   В блоке **Анкета** ответьте на вопросы о бизнесе: ассортимент, доставка, акции и прочие особенности. Ваши ответы позволят точнее настроить и таргетировать объявления. Вопросы анкеты зависят от выбранных видов деятельности.
2. Создайте дополнительные объявления.

   Можно добавить до 20 дополнительных объявлений — сгенерировать с помощью нейросети или создать самостоятельно:

   - Придумайте [заголовок и текст](ru/efficiency/text-and-title) объявления.
   - Добавьте [изображение](ru/efficiency/images). Его можно загрузить, добавить из кампаний, выбрать из предложенных нейросетью или из фотобанка. Для каждого объявления можно добавить только одно изображение.
   - Чтобы добавить [видео](ru/efficiency/video), включите опцию. Его можно загрузить или добавить из кампаний. Для каждого объявления можно добавить только одно видео.
   - Включите опцию и укажите цену товара или услуги. Так объявление станет заметнее среди аналогичных.
   - Включите опцию и укажите срок размещения объявления — период, в который будет показано объявление.

## Статистика кампании

Через несколько дней после запуска на странице кампании на вкладке **Главная** появится блок **Статистика**. На графиках можно анализировать размещение по количеству кликов и конверсий:

- «Клики» — сколько было кликов на объявления в РСЯ и Поиске, сколько переходов на компанию в Картах.
- «Конверсии» — сколько было кликов по кнопке с контактом на онлайн-визитке, которые завершились целевым действием.

Увидеть статистику отдельных объявлений нельзя, только общую картину по кампании. Графики покажут, сколько раз пользователи переходили по объявлениям и сколько при этом совершили целевых действий.
