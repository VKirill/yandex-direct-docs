# Назначение ставок

Назначение ставок работает только для фраз из кампаний с [ручным управлением](../strategies/manual-strategy.md) **с показами только на поиске**. При расчете не учитываются [корректировки ставок](../impressions/bids-adjustment.md).

Ставки в сетях назначаются только автоматически.

Чтобы назначить ставку на поиске, на странице **Кампании** отметьте нужные кампании и перейдите на страницу **Ставки и фразы**. На странице будет отображен список всех условий показа, например ключевых фраз. Нажмите на значение ставки и укажите новую ставку.

## Мастер ставок

Вы можете быстро назначить ставки на поиске для большого количества ключевых фраз, даже если они относятся к разным кампаниям или группам объявлений.

Чтобы назначить единую ставку на поиске для всех выбранных фраз:

1. Выберите нужные кампании и группы объявлений.
2. На вкладке **Ставки и фразы** выберите фразы и на нижней панели нажмите **Действия** → **Мастер ставок**.
3. Введите желаемую ставку и нажмите **Сохранить**.

Ставки будут применены к выбранным фразам.
