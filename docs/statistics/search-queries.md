# Отчет «Поисковые запросы»

- [Как построить отчет](../statistics/search-queries.md#about1)
- [Условия показа, категория таргетинга и тип соответствия](../statistics/search-queries.md#about)
- [Как работать с данными отчета](../statistics/search-queries.md#howto)

В отчете собраны запросы на поисковых площадках, по которым были показаны ваши объявления. Например, запросы, которые семантически соответствуют вашей ключевой фразе.

## Как построить отчет

1. На странице кампании перейдите по ссылке **Посмотреть статистику**.
2. На странице **Статистика** перейдите на вкладку **Поисковые запросы**.

Чтобы посмотреть статистику по всем кампаниям, в меню слева выберите **Статистика** → **Статистика по всем кампаниям**. Настроить и сохранить настройки можно также, как в [Мастере отчетов](../statistics/report-wizard.md#about).

В отчете можно увидеть:

- поисковый запрос — запрос, который ввел пользователь;
- [условие показа](../statistics/search-queries.md#criteria) — критерий, по которому показано объявление;
- [категория таргетинга](../statistics/search-queries.md#category) — соответствие поискового запроса одной из категорий;
- [тип соответствия](../statistics/search-queries.md#match-type) — связь между исходной ключевой фразой и фразой, по которой показано объявление;
- подобранные фразы — фразы, которые подобрала система с помощью фильтров для товарных объявлений и объявлений для страниц каталога, семантическое соответствие.

Данные доступны только за последние 180 дней.

В строке «Другие поисковые запросы» представлены следующие данные:

- еще не обработанные запросы — запросы, которые были сделаны менее часа назад (даже если по ним были клики);
- запросы, по которым не было кликов и было незначительное количество показов (за исключением запросов, по которым клики уже были ранее или в рамках других кампаний — они указаны в отдельных строках).

## Условия показа, категория таргетинга и тип соответствия

#### Тип условия показа

| **Срез отчета** | **Что означает** | **Пример** |
| --- | --- | --- |
| Фраза | Объявление показано по запросам, в которых ключевая фраза содержится целиком. [Подробнее о ключевых фразах](../keywords/keywords.md) | Объявление с ключевой фразой *квест на день рождения* показано по запросу *проведение квеста на день рождения* |
| Автотаргетинг | Объявление показано всем, кто ищет ваши товары или услуги, на основе содержания объявления, без использования ключевых фраз. В режиме реального времени система сопоставила заголовок, текст и страницу перехода в объявлении с поисковыми запросами пользователей. Яндекс может проверить подбор по автотаргетингу на корректность только за последние 60 дней. [Подробнее об автотаргетинге](../impression-criteria/autotargeting.md) | Объявление с текстом про путешествия в космос и страницей перехода `в-космос.рф` показано по запросу *туры на марс*. |
| Условие подбора аудитории | Представляет собой комбинацию целей/сегментов Яндекс Метрики и сегментов Яндекс Аудиторий. Объявление показано только тем пользователям, которые вас интересуют. [Подробнее о подборе аудитории](../impression-criteria/retargeting-lists.md) | По ключевой фразе *купить диван* и условию ретаргетинга *посетил сайт* объявление покажется пользователю, который напечатал в поиске «купить диван» и уже посещал сайт. |
| Интересы и соцдем | Объявление показано пользователям, чье поведение в интернете принадлежит к определенной группе. Краткосрочные интересы и привычки определяются по посещенным сайтам и приложениям, тематикам поисковых запросов, маршрутам из Яндекс Карт. [Подробнее о показах по интересам и привычкам](../products-text-image-ads/user-profile.md) | Объявление о распродаже авиабилетов или летней одежды будет показано пользователям, которые собираются в отпуск. |
| Офферный ретаргетинг | Объявление показано пользователям, которые уже взаимодействовали с товарными объявлениями или каталогами на вашем сайте. | Пользователь посмотрел товар или каталог на сайте. Он увидит эту информацию в объявлении в РСЯ. |

#### Категория таргетинга

| **Срез отчета** | **Что означает** | **Пример** |
| --- | --- | --- |
| Целевые запросы | Объявление точно отвечает на запрос пользователя. | Для пользователя «Мир рюкзаков» с объявлением «Рюкзак для ноутбука» целевой запрос — *купить рюкзак для ноутбука* |
| Альтернативные запросы | Пользователь ищет продукт, который можно заменить продвигаемым. При этом объявление также может удовлетворить запрос. | Объявление «Рюкзак для ноутбука» показано по запросу *купить сумку для ноутбука* |
| Сопутствующие запросы | Запросы по продуктам, которые могут быть интересны вместе с продвигаемым товаром или услугой. | Объявление «Рюкзак для ноутбука» показано по запросу *купить ноутбук* |
| Запросы с упоминанием конкурентов | Поиск продвигаемого продукта у конкурентов. | Объявление «Рюкзак для ноутбука» показано по запросу *купить рюкзак для ноутбука Планета рюкзаков* |
| Широкие запросы | Запрос шире, чем предложение. | Объявление «Рюкзак для ноутбука» показано по запросу *купить рюкзак* |
| Не определена | Запрос не подходит ни под одну из категорий. |  |

#### Тип соответствия

| **Срез отчета** | **Что означает** | **Пример** |
| --- | --- | --- |
| Пословное | Объявление показано по запросам, целиком содержащим заданные ключевые фразы. | Объявление с ключевой фразой *квест на день рождения* показано по запросу *заказать квест на день рождения*. |
| Семантическое | Объявление показано по запросам, которые семантически уточняют фразу: содержат опечатки, синонимы, более конкретные слова, переформулировки. Яндекс может проверить подбор по семантическому соответствию на корректность только за последние 60 дней. | Объявление с ключевой фразой *приобрести квест для детей* показано по запросу *квест для детей купить*, а с ключевой фразой *где поесть* — по запросу *рестораны быстрого питания*. |

## Как работать с данными отчета

На основе данных отчета вы можете работать над ключевыми фразами, по которым идут показы ваших объявлений, например:

- добавить запросы с высоким CTR в ключевые фразы;
- добавить нерелевантные запросы в минус-фразы на кампанию и группу объявлений.

##### **Добавить запрос в ключевые фразы**

Чтобы расширить список ключевых фраз, выберите нужные запросы в отчете, нажмите кнопку **Добавить в ключевые фразы** внизу страницы.

В открывшемся окне **Добавление фраз** отредактируйте фразу, если нужно, с помощью [операторов](../keywords/symbols-and-operators.md#how-it-works), назначьте ставки на выбранные фразы или выставите единую цену, нажмите кнопку **Сохранить**.

В результате уникальные фразы из выбранных запросов будут добавлены в ключевые фразы указанной группы объявлений. Если при добавлении лимит ключевых фраз на текущую группу будет превышен, эта группа будет скопирована и ключевые фразы, которые не поместились в исходной группе, будут добавлены в новую группу.

Если добавить поисковые запросы в ключевые фразы не получается из-за превышения лимита группы объявлений на кампанию, вы можете уменьшить число добавляемых фраз, уменьшить количество фраз в исходных группах или количество групп в кампании.

##### **Включить запрос в минус-фразы**

Чтобы уточнить ключевые фразы, выберите нерелевантные запросы, по которым показываются ваши объявления, и нажмите кнопку **Добавить в минус-фразы** внизу страницы.

В открывшемся окне **Добавление минус-фраз** вы можете отредактировать выбранную фразу, например, добавить операторы `+`, `!`, `[]`. По умолчанию добавляемая фраза указана в кавычках `""`. Вы также можете выбрать уровень, где будет работать минус-фраза, — группа объявлений или кампания. Нажмите кнопку **Сохранить**.

Если при добавлении выбранных запросов в минус-фразы вы превысите лимиты на количество символов на группу или кампанию, будет выведено предупреждение.
