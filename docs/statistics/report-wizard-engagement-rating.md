# Показатели вовлеченных сессий

- [Показатели ВС](ru/statistics/report-wizard-engagement-rating#parameters)
- [Применение в анализе кампаний](ru/statistics/report-wizard-engagement-rating#usage)
- [Прочерки в отчете](ru/statistics/report-wizard-engagement-rating#dashes)

Показатели вовлеченных сессий (ВС) оценивают вероятность целевых действий (покупки, заявки, подтверждение регистрации) на продвигаемом сайте.

Статистика по показателям ВС доступна с 14 июля 2017 года для всех кампаний, в которых продвигается сайт с установленным счетчиком Метрики.


## Показатели ВС

#### Сумма ВС

Совокупная оценка конверсионности визитов из Директа. Показатель отражает заинтересованность в товаре или услуге пользователей, кликнувших по объявлению.

С помощью алгоритмов машинного обучения система анализирует сессию пользователя, который перешел по объявлению, и предсказывает вероятность целевых действий на сайте. Прогноз строится на данных о посещаемых страницах сайта, глубине и времени просмотра.

Конверсионность каждой сессии оценивается в баллах от 0 до 1. Баллы по выбранным визитам суммируются. Их суммарное количество может быть дробным.

#### Цена ВС

Усредненная стоимость конверсии, отношение расходов на продвижение к оценке конверсионности визитов (Расход/Сумма ВС).

#### Доля ВС (%)

Эффективность конверсии, отношение оценки конверсионности визитов к общему количеству переходов по объявлениям (Сумма ВС/Клики × 100%).

## Применение в анализе кампаний

Показатели помогут оценить эффективность кампаний, когда у вас не настроены цели в Метрике или созданные цели не дают представления о результатах работы объявлений.

Если вы используете цель в Метрике для анализа размещения, сравните данные по ней и показателям ВС. В случае, когда по цели трафик в срезе конверсионный, а по показателям ВС нет (или наоборот), проверьте настройки кампании, структуру сайта, цель, по которой измеряется эффективность.

Чтобы узнать, какие форматы объявлений, аудитории, площадки, регионы показов приносят больше целевых визитов, сравнивайте в разных срезах относительные показатели — Цену ВС и Долю ВС (%).

Регулярный анализ показателей ВС поможет понять, что изменить в кампании или на сайте для улучшения работы объявлений.

> Например, для вашей кампании в сетях идут показы объявлений в разных форматах.  Сравните статистику по форматам. Данные по Цене ВС и Доле ВС (%) позволяют сделать вывод, что видео эффективно работают для привлечения целевой аудитории. Настройте повышающие [корректировки ставок](ru/impressions/bids-adjustment#video-extension-bids-adjustment) для объявлений с видео, чтобы увеличить вероятность их показов.

## Прочерки в отчете

В столбцах отчета будут прочерки, если:

- выбранные срезы отчета содержат недостаточное количество кликов для оценки конверсионности визитов. Увеличьте период просмотра статистики, измените группировку данных, выбранные срезы и фильтры;
- в параметрах кампании не указан счетчик Метрики;
- на сайте не установлен счетчик Метрики;
- анализируется кампания с типом **Продвижение приложений**;
- накопленной статистики по кампании недостаточно для расчета прогноза.
