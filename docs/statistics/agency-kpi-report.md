# Отчет для агентств по KPI

- [Как заказать отчет](../statistics/agency-kpi-report.md#how-to-order)
- [Данные в отчете](../statistics/agency-kpi-report.md#data-availability)

Отчет по KPI содержит информацию о функциональностях, которые должны быть подключены в кампаниях для получения премии. Отчет можно получить за период, не включающий последние 3 дня от текущей даты.

## Как заказать отчет

Получить отчет можно на странице **Отчёт по KPI**. Выберите период длительностью не более 3 месяцев и нажмите кнопку **Заказать отчёт**.

Отчет формируется в фоновом режиме. Время подготовки отчета зависит от количества данных и занимает от 15 минут до нескольких часов. Когда подготовка отчета завершится, он получит статус «Готов». Скачайте отчет в формате XLSX.

Имя отчета генерируется в формате `{dateFrom}-{dateTo}-{login}-qkpi-report_{randID}.xlsx`, где:

`{dateFrom}` — начальная дата отчетного периода;

`{dateTo}` — конечная дата отчетного периода;

`{login}` — логин представителя агентства, по которому построен отчет;

`{randID}` — 4 случайных символа.

## Данные в отчете

Внимание

Суммы по оборотам в незакрытом периоде (ориентировочно до середины следующего месяца) могут не соответствовать выставленным после закрытия актам.

Отчет содержит две вкладки: на одной выгружаются данные по performance-кампаниям, на другой#160;— по медийным кампаниям.

Для каждой кампании указана информация о функциональностях, которые должны быть подключены в кампаниях для получения премии.

Полный список таких функциональностей фиксируется в условиях договора и может быть изменен.

**Номер договора**

:   Договор, по которому работает агентство.

**ID клиента**

:   Идентификатор клиента.

**Логин клиента**

:   Логин клиента, указанный при регистрации.

**Название клиента**

:   Название клиента, указанное в интерфейсе Директа.

**Логин представителя**

:   Логин представителя по работе с клиентами, которому назначен клиент. Если клиент не имеет представителя — логин главного представителя агентства.

**Номер РК в Директе**

:   Кампания, по которой выгружаются данные.

**Название РК**

:   Наименование кампании, которое указано в интерфейсе Директа.

**Оборот РК всего**

:   Стоимость оказанных услуг в  рамках кампании за выбранный период без НДС.

**Домен**

:   Адрес интернет-страницы (или несколько адресов), на которую ведет ссылка из рекламно-информационных материалов клиентов агентства.
