# Отчет по актам для агентств

- [Как скачать отчет](ru/statistics/agency-acts-report#how-to-order)
- [Данные в отчете](ru/statistics/agency-acts-report#data-availability)

С 1 мая 2024 для агентств станет доступен **Отчет по актам**, с помощью которого можно будет сопоставить данные из статистики с данными в актах и спрогнозировать итоговую сумму в акте по окончании месяца.

Для пользователей доступны два источника данных о тратах на продвижение:

1. Статистика — данные в Мастере отчетов, строятся по времени события (когда произошел показ, клик или конверсия).
2. Акты — бухгалтерский документ, выставляемый по окончании отчетного периода. По актам фиксируется расход денег на продвижение.

При этом в акте конверсия учитывается в день фактического списания денег, а в статистике расход по конверсии привязывается ко дню клика. Если клик был в конце января, а конверсия в начале февраля, то расход по ней запишется в акт за февраль, а в статистике — за январь.

Также иногда возникают вопросы о компенсациях за фродовые конверсии и клики, которые возвращаются на счет пользователя. Из-за них акт может быть выставлен на сумму меньшую, чем реальный расход.

Чтобы соотнести данные в актах и Мастере отчетов, загрузите **Отчет по актам**. В отчете показывается, сколько было потрачено по конкретной кампании за месяц с детализацией по дням и по видам трат: кешбеки, промокоды, корректировки статистики, реальные деньги. Отчет строится по времени актирования — фактического списания реальных денег и бонусных средств с пользователя на основе конечного акта за месяц. Расходы на продвижение в отчете будет полностью сходиться с суммой по акту.

Отчет можно строить и ежедневно, и следить за динамикой трат реальных денег, а не только за общей статистикой. Но по-дневные расходы — это приблизительные данные, так как в течение месяца расход на определенную дату может измениться.

## Как скачать отчет

Чтобы скачать отчет, в меню слева выберите **Статистика** → **Мастер отчетов**. В [Мастере отчетов](ru/statistics/report-wizard.dita):

1. Выберите период, за который надо построить отчет. Выбор прочих настроек, срезов и столбцов на данные в отчете не влияет.
2. Нажмите **Показать**.
3. В верхнем правом углу страницы нажмите **Отчет по актам.xlsx**.

## Данные в отчете

Отчет предоставляет подневную детальную статистику по каждой кампании. В зависимости от выбранной стратегии в настройках, для каждой кампании отображается количество действий, подлежащих оплате: в колонке «Показы», «Клики» или «Конверсии». В колонке «Всего» указана общая сумма расходов, а в следующих колонках идет детализация того, из каких типов средств она состоит.

Итоговая сумма за месяц в отчетах статистики должна быть примерно равна итоговой сумме в отчете по актам, но никогда не будет совпадать из-за перетоков событий между датами, отложенных событий (конверсий, кликов, учета недействительных кликов и конверсий), разной аллокацией событий по времени.

Итоговая сумма по столбцу «Деньги» отчета по актам, построенного за месяц, должна быть равна итоговой сумме расхода в акте.
