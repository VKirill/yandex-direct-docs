Как оценить работу стратегии с несколькими целями

# Как оценить работу стратегии с несколькими целями

Советы в статье применимы, если в кампаниях настроена одна из стратегий с несколькими целями:

- [Максимум конверсий](ru/strategies/average-cpa) с оплатой за клики и ограничением расхода по средней цене конверсии.
- [Максимум конверсий](ru/strategies/average-cpa) с оплатой за конверсии и ограничением расхода по цене.

Оптимизация по нескольким целям полезна в следующих случаях:

- В кампании используются одинаково важные для бизнеса цели, например заявка, звонок или сообщение в чат. Оптимизация по ним поможет привлечь пользователей, которые достигают любую из них.
- В кампаниях по воронке конверсий, где указана ключевая цель и шаги на пути к ней, оптимизация по нескольким целям даст стратегии больше данных для обучения. Например, главной целью может быть офлайн-конверсия — оплата заказа, а шагами на пути к ней — оформление заказа на сайте и добавление товара в корзину. Дополнительные данные помогут стратегии определить, какие действия пользователей приводят к главной цели, а их цена — насколько одни конверсии важнее других.

Если вы настроили оптимизацию по нескольким целям, при **оплате за конверсии бюджет будет списываться за каждое достижение цели в рамках одного визита**. Если пользователь достиг две разные цели в рамках одного визита, оплачиваться будет каждое достижение. Если одна и та же цель достигнута несколько раз за визит, то будет одно списание на сумму указанной цены.

> Например, вы добавили две цели: «Оформление заказа» и «Оплата заказа». Если пользователь оформил заказ на сайте и оплатил его при получении, спишется стоимость каждой из конверсий.

Поэтому рассчитайте стоимость конверсий так, чтобы при достижении обеих целей расходовалась только целевая стоимость оплаченного заказа. Вы также можете использовать оплату за клики, когда оптимизируете кампанию на цели из одной воронки.

## Проведите комплексный анализ

При оптимизации по нескольким целям с разной ценой (целевой CPA) бюджет распределяется между разными целями. Поэтому, если разделить все расходы на количество конверсий по одной из целей, например, только главной для бизнеса, можно получить искаженный результат. Чтобы оценить, выдерживает ли стратегия ограничения по цене конверсии, рекомендуем ориентироваться не на стоимость конверсии для каждой цели, а на общие результаты стратегии.

В кампаниях со стратегией Максимум конверсий проведите комплексный анализ одним из способов:

- [Сравнение средневзвешенного целевого CPA с фактическим](ru/statistics/multiple-goals-strategy#value)
- [Сравнение ожидаемого расхода с фактическим](ru/statistics/multiple-goals-strategy#expenses)

#### Сравнение средневзвешенного целевого CPA с фактическим

Этот способ позволяет оценить, насколько ожидаемая цена конверсии отличается от фактической.

Например, в кампании получено 20 конверсий по цели «Оформление заказа», средняя цена конверсии 100 рублей, и 5 конверсий по цели «Оплата заказа» со средней ценой конверсии 300 рублей.

1. Рассчитайте средневзвешенную цену конверсии (wtCPA).

   wtCPA = (цена цели X×количество конверсий X)+(цена цели Y×количество конверсий Y)количество конверсий X+количество конверсий Y\frac{(цена\ цели\ X\times количество\ конверсий\ X) + (цена\ цели\ Y\times количество\ конверсий\ Y)}{количество\ конверсий\ X + количество\ конверсий\ Y}количество конверсий X+количество конверсий Y(цена цели X×количество конверсий X)+(цена цели Y×количество конверсий Y)​

   wtCPA = (100 рублей×20)+(300 рублей×5)20+5\frac{(100\ рублей\times 20) + (300\ рублей\times 5)}{20 + 5}20+5(100 рублей×20)+(300 рублей×5)​ = 2000+150025\frac{2000 + 1500}{25}252000+1500​ = 140 рублей.
2. В Мастере отчетов постройте отчет за тот же период и уточните данные по фактическому СРА. Используйте настройки отображения метрики СРА суммарно по всем целям.

   Фактический CPA = расходколичество конверсий X+количество конверсий Y\frac{расход}{количество\ конверсий\ X + количество\ конверсий\ Y}количество конверсий X+количество конверсий Yрасход​
3. Сравните средневзвешенную целевую цену конверсии и фактическую.

   Если фактический CPA меньше wtCPA (например, фактический CPA составил 136 рублей при wtCPA в 140 рублей) — стратегия укладывается в ограничения.

Примечание

На фактическую среднюю цену конверсии влияют выбранные [корректировки](ru/impressions/bids-adjustment). Учитывайте их в сравнении.

Если у всех целей одинаковый целевой CPA, то wtCPA будет равен целевому CPA. В этом случае разделите расход на общее количество конверсий.

Если достигнута только одна из нескольких целей, например цель X, wtCPA будет равен целевому CPA для цели X. В этом случае можно ориентироваться на CPA данной цели.

#### Сравнение ожидаемого расхода с фактическим

Этот способ также помогает проверить, что ожидаемый расход не превышает фактический. Здесь не вычисляется конкретное значение, а сравнивается общая ценность конверсий с расходами на продвижение.

1. Рассчитайте ожидаемый расход на полученные конверсии:

   Ожидаемый расход = (конверсии X \* целевой CPA X) + (конверсии Y \* целевой CPA Y)
2. Сравните фактический расход и ожидаемый.

   Если фактический расход меньше ожидаемого, стратегия выдерживает ограничения.

Примечание

На фактическую среднюю цену конверсии влияют выбранные [корректировки](ru/impressions/bids-adjustment). Учитывайте их в сравнении.

В Мастере отчетов метрика «Ценность конверсий» для стратегии «Максимум конверсий» рассчитывается как ожидаемый расход. Поэтому можно сравнивать общую «Ценность конверсий» с фактическим расходом.

Например, если общая ценность конверсий по целям «Оформление заказа» и «Оплата заказа» составила 3500 рублей (2000 рублей + 1500 рублей), а расход составил 3400 рублей, заданные в стартегии ограничения выдерживаются.
