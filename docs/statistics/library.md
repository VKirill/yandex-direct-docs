# Библиотека отчетов

В библиотеке вы можете выбрать готовый отчет с уже подобранными настройками для разных типов кампаний и задач аналитики. Например, для перфоманс-кампаний настройки помогают отследить количество конверсий, расходы, стоимость целевого действия, доход и ДРР. Для охватных кампаний — медийные показатели охватов целевой аудитории: просмотры, видимость, среднюю стоимость просмотра.

С помощью готовых отчетов можно не только оценить текущие результаты продвижения, но и находить новые идеи для более глубокой аналитики конкретных показателей.

Чтобы перейти в библиотеку, в разделе **Статистика** выберите **Библиотека отчётов**.

Готовый отчет можно изменить — добавить нужные группировки и метрики, а затем сохранить настройки. При следующем открытии отчет загрузится уже с вашими изменениями. Или вы можете сохранить отчет как новый — в этом случае он появится на вкладке **Мои отчёты**. Если с аккаунтом работает несколько человек, можно выбрать, кому будет доступен отчет — только вам или всем пользователям.

После сохранения отчет с вашими настройками будет открываться также и из интерфейса при переходе к статистике по новому Мастеру отчетов: например, адаптированная версия отчета по перфоманс-кампаниям будет доступна по ссылкам **Статистика (новая)** или **Статистика - Новый интерфейс** из списка кампаний или на странице редактирования перфоманс-кампании.

Примечание

Некоторые показатели в готовых отчетах недоступны для редактирования. Если необходимо изменить такие показатели, [создайте](ru/statistics/library#create) новый отчет с нужными настройками.
