Мастер отчетов — это инструмент для просмотра статистики, который позволяет анализировать эффективность кампаний и получать подробную информацию о показателях их работы. В Мастере отчетов можно создавать кастомные отчеты — использовать комбинации разных срезов данных и показателей в зависимости от цели.

Доступен в левом меню интерфейса Директ Про в разделе **Статистика**, при редактировании кампании и на общей странице кампаний.

## Как создать отчет

Создать новый отчет можно одним из способов:

- В левом меню выберите **Статистика** → **Новый Мастер отчётов** или **Статистика** → **Мои отчёты** → **Создать новый отчёт**.
- На странице редактирования кампании нажмите  справа от названия кампании и в выпадающем списке выберите **Статистика (новая)**.
- На странице кампаний нажмите  справа от нужной кампании и в выпадающем списке выберите **Статистика (новая)**. Этот способ недоступен для кампаний, созданных через Мастер кампаний.
- На странице кампаний выделите одну или несколько кампаний и нажмите **Статистика 2 кампаний (новая)** в контекстном меню в нижней части экрана или в списке, выпадающем по кнопке **Действия**. Этот способ недоступен для кампаний, созданных через Мастер кампаний.

Новый Мастер отчетов можно выбрать инструментом по умолчанию — тогда все ссылки на статистику будут вести на него. Для этого откройте новый Мастер отчетов, в правом верхнем углу нажмите  и выберите **Использовать как интерфейс по умолчанию**. Выбор можно изменить.

## Настройки отчета

### Период

Задайте период вручную или выберите один из предложенных. Все доступные периоды находятся в выпадающем списке.

Если вам нужно включить в отчет данные за сегодняшний день, отметьте пункт **Включая сегодня**. Например, если 10 апреля запросить данные за последние 7 дней с учетом сегодняшнего дня, отчет будет включать данные за период с 4 по 10 апреля. Без учета сегодняшнего дня — с 3 по 9 апреля.

Основные данные в отчете доступны за последние три года.

Полные данные по поисковым запросам доступны за последние шесть месяцев. Информация за более длительный период может быть неполной — количество показов, кликов и другие метрики будут точные, но сам текст запроса для некоторых аукционов может быть не определен.

Пример

|  |  |  |  |
| --- | --- | --- | --- |
| **Тип площадки** | **Поисковый запрос** | **Показы** | **Клики** |
| Поиск | купить чайник | 1 000 | 100 |
| Поиск | купить сковородку | 900 | 80 |
| Поиск | – | 500 | 40 |

**Сравнение периодов**

При анализе данных можно не только посмотреть статистику за конкретный период, но и сравнить два периода в рамках одного отчета.

Чтобы сравнить периоды, задайте первый период, нажмите **Сравнение** и выберите второй период для сравнения:

- Предыдущий равный период.
- Предыдущая неделя.
- Предыдущий месяц.
- Предыдущий год.

Также указать период для сравнения можно вручную по кнопке **Указать период**. Даты первого и второго периодов при настройке сравнения могут пересекаться.

Чтобы поменять местами указанные периоды, нажмите  между периодами.

Настройки сравнения учитываются при сохранении отчетов. Например, можно создать отчет по сравнению «неделя к неделе», сохранить его и открывать из раздела **Мои отчёты**.

Для более удобной работы с большими отчетами можно закрепить колонку, чтобы необходимые данные отображались в отчете при горизонтальной прокрутке. Для этого справа от названия колонки нажмите  и выберите **Закрепить колонку**.

Посмотреть видео. Сравнение периодов

Для отображения динамики изменений можно выбрать цвет: красный или зеленый. Вы можете выбрать, каким цветом отмечать позитивные и негативные изменения.

Настроить цветовое форматирование можно как для всей таблицы, выбрав нужные параметры, так и для отдельных колонок. Настройки автоматически сохраняются для всех отчетов, при необходимости вы сможете их изменить.

### Аккаунты

В новом Мастере отчетов можно строить отчеты по нескольким аккаунтам, если вы работаете через Управляющий аккаунт. Для этого выберите аккаунты и укажите нужные кампании.

Примечание

Настройка **Аккаунты** в интерфейсе нового Мастера отчетов доступна только при работе через Управляющий аккаунт.

Для Управляющего аккаунта доступна группировка **Аккаунт**. Ее можно добавить в отчеты, чтобы получить данные по разным аккаунтам, например, расходы.

### Группировки и метрики

Метрики и группировки разделены на смысловые группы. Доступен поиск по названиям. Все группировки можно развернуть.

В группировках выберите данные, для которых нужно построить отчет, а в метриках — нужные статистические показатели. Для построения отчета выберите хотя бы одну метрику.

В меню настроек справа отображаются выбранные группировки и метрики. Вы можете изменить их порядок, перетаскивая элементы за значок двунаправленной стрелки, или удалить ненужные.

Некоторые группировки, в том числе и фильтры по ним, могут быть несовместимы между собой или с конкретными метриками. В этом случае вы увидите подсказку с перечнем несовместимых элементов.

### Цели и атрибуция

Выберите цели, по которым хотите рассчитать конверсионные метрики (Конверсии, CPA и другие). Вы можете воспользоваться поиском по названиям или номерам целей и счетчиков. Если не выбрано ни одной цели, по конверсионным метрикам будет отображаться количество целевых визитов, в которых была достигнута хотя бы одна из целей.

Настройте отображение статистики по конверсионным метрикам. Выберите **Суммарно по всем выбранным целям** или **Отдельно по каждой цели**, либо оба варианта. Для отображения колонок отдельно по каждой цели выберите до 20 целей. Если выбрано более 20, отобразить можно только общее количество конверсий по всем выбранным целям.

Подробнее о [конверсиях в отчете](ru/statistics/performance-stat-guide).

Пользователи могут переходить на сайт несколько раз, например по объявлению, по ссылке в результатах поиска или по закладке, сохраненной в браузере. Атрибуция — правило, согласно которому определяется, какой из переходов на сайт считается источником визита.

Выберите модель атрибуции, по которой будет показываться статистика.

Подробнее о [моделях атрибуции](ru/statistics/attribution-model).

### Фильтры

Фильтры позволяют отсекать ненужные данные и фокусироваться на наиболее важных.

По умолчанию в настройках установлен фильтр, который исключает архивные кампании, если аккаунту больше полугода. Это ускоряет создание отчетов в больших аккаунтах.

**Как работают сочетания фильтров**

- Фильтры работают по принципу пересечения блоков между собой по логическому оператору «И».
- Если внутри фильтров выбрано несколько вариантов:

  - для операторов «Равно» и «Содержит» применяется логический оператор «ИЛИ»;
  - для операторов «Не равно» и «Не содержит» — логический оператор «И».
- Фильтры по текстовым данным, например по названию кампании, можно выбрать несколько раз, чтобы составить сложные условия. Текстовый фильтр с оператором «Содержит» также можно выбрать несколько раз.

Пример

|  |  |  |
| --- | --- | --- |
| **Условие для фильтра** | **Названия кампаний** | **Результат фильтрации** |
| «содержит бренд ИЛИ брэнд»  И  «содержит Москва ИЛИ мск»  И  «не содержит промо И акция» | - РФ\_Бренд\_Промо\_весна\_2025 - РФ\_Бренд\_Основная\_весна\_2025 - МСК\_Бренд\_Промо\_весна\_2025 - МСК\_Бренд\_Основная\_весна\_2025 - МСК\_Целевые\_Основная\_весна\_2025 - рф\_брэнд\_промо\_лето\_2025 - рф\_брэнд\_основная\_лето\_2025 - мск\_брэнд\_промо\_лето\_2025 - мск\_брэнд\_категории\_лето\_2025 - мск\_целевые\_основная\_лето\_2025 - РФ\_бренд\_акция\_осень\_2025 - РФ\_бренд\_основная\_осень\_2025 - москва\_бренд\_акция\_осень\_2025 - москва\_бренд\_основная\_осень\_2025 | - МСК\_Бренд\_Основная\_весна\_2025 - мск\_брэнд\_категории\_лето\_2025 - москва\_бренд\_основная\_осень\_2025 |

**Другие особенности работы фильтров**

- В фильтре по кампаниям, а также в числовых фильтрах, таких как фильтры по номеру группы, в поле ввода можно вставить список ID, перечисленных через разделитель (запятая или перенос строки).
- Для фильтров с оператором «содержит» можно использовать специальные символы:

  `^` — соответствует одному любому символу. Например, под условие фильтра «Название кампании содержит бр^нд» попадут кампании с названиями «Поиск\_бренд\_Мск» и «РСЯ\_брэнд\_РФ».

  `*` — соответствует любому количеству любых символов. Например, под условие фильтра «Название кампании содержит Промо\*Поиск» попадут кампании с названиями, «Директ\_Промо\_МСК\_Поиск», «Промо\_РФ\_Поиск\_2».

## Предустановленные настройки

В новом Мастере отчетов по умолчанию установлены базовые настройки. Чтобы изменить их, задайте нужные цели, метрики, группировки и другие настройки. Чтобы сохранить изменения, в правом верхнем углу нажмите **Сохранить отчёт** → **Сохранить настройки отчёта**. Настройки сохранятся на уровне вашего логина. У других представителей с доступом к данному кабинету Директа предустановленные настройки не изменятся.

## Меню быстрых действий

В новой версии инструмента для каждой колонки отчета доступно меню быстрых действий. Оно позволяет:

- выводить данные в абсолютных значениях или процентах;
- перемещать колонки;
- сортировать строки по возрастанию или убыванию показателя;
- добавлять фильтры на метрики;
- удалять колонки или менять их ширину;
- закреплять колонку с группировкой.

## Как сохранить и отредактировать отчет

Создавайте разные версии отчетов под конкретные задачи и делитесь ими.

Чтобы сохранить отчет, в правом верхнем углу нажмите **Сохранить отчёт** → **Сохранить отчёт как новый**. При сохранении вы можете выбрать, кому отчет будет доступен: только вам или всем представителям аккаунта.

Сохраненные отчеты можно посмотреть и отредактировать в в левом меню в разделе **Статистика** → **Мои отчёты**. Чтобы дублировать отчет, переименовать его или удалить, рядом с названием нажмите  и выберите нужное действие.

Чтобы отредактировать отчет, откройте его, внесите изменения и в правом верхнем углу нажмите **Сохранить отчёт** → **Сохранить настройки отчёта**.

Если вы хотите создать новый отчет на основе предыдущего:

- Откройте отчет и внесите изменения.
- В правом верхнем углу нажмите **Сохранить отчёт** и в выпадающем списке выберите **Сохранить отчёт как новый**.

Любой отчет можно переименовать или удалить.

Если отредактировать и сохранить отчет, который доступен для всех представителей аккаунта, это изменение отразится у всех. Аналогично при переименовании и удалении отчета.

Важно

Если отчет не открывался в течение года, он автоматически удалится.

### Поделиться отчетом

Чтобы поделиться отчетом, в правом верхнем углу нажмите **Поделиться** и в открывшемся окне скопируйте ссылку. Там же будет указан срок ее действия. Отчет будет доступен пользователям, у которых есть доступ к вашему аккаунту.

Примечание

Если скопировать ссылку из адресной строки браузера, поделиться отчетом с другими пользователями не получится — доступ к отчету будет только у вас. Кроме того, срок действия такой ссылки будет короче.

### Скачать отчет

Отчет можно скачать в форматах XLSX и CSV.

Чтобы настроить формат файла для скачивания, в правом верхнем углу отчета нажмите значок  и выберите формат: **XLSX**, **CSV** или оба варианта.

Чтобы скачать файл:

1. В правом верхнем углу отчета нажмите кнопку **Скачать**. Если в настройках установлен один формат файла, загрузка начнется автоматически.
2. Если в настройках установлены оба формата файла, выберите вариант для скачивания.

## Управление показами на площадках РСЯ

Внимание

Исключение площадок и сетей может привести к неэффективной работе кампании и потере дополнительной аудитории, которую вы могли бы привлечь.

Вы можете управлять показами на конкретных площадках Рекламной сети Яндекса, используя новый Мастер отчетов. Площадки можно запретить или разрешить сразу для нескольких кампаний в рамках одного отчета.

Чтобы запретить или разрешить показы:

1. В новом Мастере отчетов нажмите **Группировки и метрики**. На вкладке **Группировки** выберите **Площадки** → **Название площадки**. Либо в [Библиотеке отчетов](ru/statistics/library) в блоке **Оптимизация размещения** выберите **Площадки**.
2. Для отображения детализации по кампаниям в группировках также выберите **Кампании** → **№ кампании** и/или **Название кампании**.
3. Если чек-боксы выбора площадок не доступны, в колонке **Название площадки** нажмите на  и включите опцию **Режим массовых действий**. Чтобы убрать чек-боксы, отключите опцию.
4. Отметьте нужные площадки в нужных кампаниях.
5. В меню снизу нажмите **Запретить показы** или **Разрешить показы**.

Площадки, на которых запрещены показы, в новом Мастере отчетов выделены серым цветом.

Чтобы посмотреть изменения, связанные с запретом или разрешением показов на плошадках, в меню слева нажмите **Инструменты** → **История изменений**.

## Добавление минус-слов и минус-фраз

Вы можете добавлять минус-слова и минус-фразы на уровне кампаний и групп, используя новый Мастер отчетов.

Чтобы дополнить список минус-слов и минус-фраз:

1. В новом Мастере отчетов нажмите **Группировки и метрики**. На вкладке **Группировки** выберите **Таргетинги** → **Поисковые запросы** → **Поисковый запрос**. Либо в [Библиотеке отчетов](ru/statistics/library) в блоке **Оптимизация размещения** выберите **Поисковые запросы**.

   Также в отчете должна быть добавлена группировка по кампании или по группе. Нажмите **Применить**.

   Совет

   Добавьте в отчет группировку по группе, тогда вы сможете выбрать, на каком уровне добавить минус‑фразу — на кампанию или на группу.
2. Если чек-боксы для выбора поисковых фраз не доступны, в колонке **Поисковый запрос** нажмите на  и включите опцию **Режим массовых действий**. Чтобы убрать чек-боксы, отключите опцию.
3. Отметьте нужные фразы.
4. В меню снизу нажмите **Добавить в минус-фразы**. В открывшемся окне фразы можно отредактировать — убрать лишние слова или добавить операторы.

## Доход и ценность конверсий

В новом Мастере отчетов доступны метрики, которые помогают отслеживать результаты продвижения в Директе с учетом бизнес-показателей — «Доход» и «Ценность конверсий».

Метрика «Доход» теперь рассчитывается только по выбранным целям. В боковом меню перейдите в раздел **Конверсии** → **Настройка метрики «Доход»** и укажите цели, по которым будет рассчитываться доход и производные от него метрики: ДРР, ROI и прибыль. Если вы выбрали несколько целей, доход будет рассчитан как сумма по всем целям. Это могут быть цели, данные по которым передаются из Метрики, мобильных трекеров или офлайн-конверсии. Цель, по которой не передается доход, не будет включена в расчет, пока данные не начнут поступать в Директ.

Метрика «Ценность конверсий» отражает вашу оценку: какую экономическую выгоду принесет достижение целей. Также доступны производные от нее метрики: ДРР, ROI и прибыль по ценности. Чтобы передать ценность, укажите нужное значение в настройках стратегии кампании, а также с помощью Метрики, мобильных трекеров или офлайн-конверсий.

Подробнее о [доходе и ценности конверсий](ru/statistics/profit-and-value).
