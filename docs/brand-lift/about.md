## Этап 1. Поиск аудитории

В ходе исследования опрашивается реальная аудитория кампании. Приглашения пройти опрос получают более 10 000 пользователей. Половина их них — те, кто видел рекламу (тестовая группа), и столько же — в контрольной группе. Далее из них формируется выборка, на которой и проводится аналитика. Для создания контрольной группы используется [виртуальный аукцион](../research.md#virtual-auction).

Обе группы отбирают из целевой аудитории кампании, но контрольная группа вместо рекламы видит опрос.

## Этап 2. Вовлечение

Прежде чем пройти опрос, его потенциальные участники видят приглашение поделиться опытом. Это позволяет вовлекать аудиторию, которая готова поделиться своим мнением прямо сейчас.

Это важно, так как, по данным опроса Взгляда, треть пользователей проходят короткие опросы, потому что убеждены, что их ответы могут быть полезными, и ещё пятая часть — потому что хотят поделиться мнением.

## Этап 3. Опрос

Пользователям из обеих групп, принявшим приглашение, предлагается ответить на три вопроса. Добровольное участие — базовое условие достоверности исследования, поэтому прекратить опрос можно в любой момент. В таком случае анкета не будет засчитана.

## Какие вопросы можно задать в ходе исследования

Brand Lift

Brand Lift с Top of Mind

Вы можете выбрать три из шести доступных вопросов. В каждом из них исследуется один показатель эффективности кампании. Формулировки вопросов фиксированные, но есть возможность заменить слова, выделенные курсивом. Рекомендуем обязательно замерять запоминаемость рекламы (Ad Recall) — эта метрика отражает прямое влияние на аудиторию.

|  |  |
| --- | --- |
| **Показатель** | **Вопрос** |
| Brand Awareness (узнаваемость бренда) | Какие *бренды* вам знакомы? |
| Ad Recall (запоминаемость рекламы) | Рекламу каких *брендов* вы видели за последние две недели? |
| Purchase Intent (намерение совершить покупку) | Какой *бренд* вы купите в следующий раз?  Где вы *купите услуги* в следующий раз? |
| Product Consideration (рассмотрение продукта для покупки) | *Какие бренды* вы готовы рассмотреть? |
| Brand Favorability (предпочтение бренда) | Какие *бренды* вы предпочитаете? |
| Ad Message Recall (запоминаемость рекламного слогана) | Какие рекламные слоганы вы знаете? |

Чтобы упростить восприятие вопросов, можно незначительно их редактировать, но только так, чтобы не нарушать их логику.

Вы можете выбрать три из шести доступных вопросов. В каждом из них исследуется один показатель эффективности кампании. Формулировки вопросов фиксированные, но есть возможность заменить слова, выделенные курсивом. Квадратными скобками выделено место, куда будет подставлена категория, выбранная при настройке исследования.

|  |  |  |  |
| --- | --- | --- | --- |
| **Показатель** | **Обязательный/необязательный выбор** | **Вопрос** | **Тип вопроса** |
| Top of Mind Ad | Обязательно выбрать только один из двух показателей. | Рекламу каких [брендов из этой категории] вы видели за последние две недели? | Открытый, пользователи сами вписывают названия брендов. |
| Top of Mind Brand | Какие [бренды из этой категории] вы знаете? |
| Brand Awareness (узнаваемость бренда) | Обязательно выбрать один из двух показателей, можно выбрать оба. | Какие *бренды* вам знакомы? | Закрытый, пользователи выбирают бренды из предложенных вариантов. |
| Ad Recall (запоминаемость рекламы) | Рекламу каких *брендов* вы видели за последние две недели? |
| Purchase Intent (намерение совершить покупку) | Дополнительные показатели.  Можно выбрать один, если выше выбрано два из трех показателей. | Какой *бренд* вы купите в следующий раз?  Где вы *купите услуги* в следующий раз? |
| Product Consideration (рассмотрение продукта для покупки) | *Какие бренды* вы готовы рассмотреть? |
| Brand Favorability (предпочтение бренда) | Какие *бренды* вы предпочитаете? |
