# Как проводится Visit Lift

В основе исследования — анализ визитов целевой аудитории на сайт. Для этого оценивается реальная аудитория рекламной кампании.

Из целевой аудитории кампании отбирается тестовая (90%) и контрольная (10%) группа. Тестовая группа включает пользователей, которые видели рекламу. Контрольная группа состоит из пользователей той же целевой аудитории, которым реклама намеренно не демонстрируется. Для создания контрольной группы используется [виртуальный аукцион](../research.md#virtual-auction).
