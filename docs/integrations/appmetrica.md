# Пиксель AppMetrica для мобильных приложений

В медийных кампаниях для рекламы мобильных приложений настройте пиксель AppMetrica. Он помогает отслеживать, как пользователи после просмотра медийной рекламы совершали установки приложения и события в нем, а также сколько конверсий принесла медийная реклама в мобильных приложениях.

### Преимущества

- Больше данных о медийной рекламе в приложениях.
- Новые возможности для аналитики медийной рекламы для приложений.

### Настройка

Для отслеживания post-view и post-click установок и событий настройте треккинговую ссылку и добавьте ее в объявление:

1. Интегрируйте SDK AppMetrica в ваше приложение.
2. Настройте трекинговую ссылку. Подробнее об [установке пикселя AppMetrica](https://appmetrica.yandex.ru/docs/ru/mobile-tracking/yandex-direct-media-tracking) в Справке AppMetrica. Установка пикселя не требует дополнительной оплаты.
3. Создайте рекламную кампанию. Для рекламы мобильных приложений можно использовать [Медийные баннеры](ru/products-cpm-campaign/create), [Видеорекламу](ru/products-cpm-campaign-video).
4. В поле **Ссылка в объявлении** укажите трекинг-ссылку из AppMetrica, а в поле **Счётчик показов** — ссылку **Impression URL** из AppMetrica.
5. Запустите рекламную кампанию.

Дополненная статистика по медийной кампании (post-view и post-click установок и событий) доступна только в AppMetrica.

Примечание

Сейчас AppMetrica помогает отслеживать только тех пользователей, которые после просмотра или клика на рекламу совершили установку приложения. По этим же пользователям в дальнейшем будет отображено совершение событий внутри приложения.

Если приложение уже было установлено у пользователя на момент показа или клика на рекламу, то его события в приложении не будут атрибуцированы к рекламе, и статистика по ним не будет отображаться в отчетах по этой рекламной кампании. В будущих обновлениях эта возможность будет поддержана.
