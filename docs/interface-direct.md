# Интерфейс Директ

**Директ** – базовый интерфейс настройки кампаний для начинающих пользователей Директа. Он подойдет предпринимателям, которые хотят продвигать свой бизнес в Поиске, Рекламной сети, Товарной галерее, Яндекс Картах. Для этого доступны два инструмента:

- [Простой старт](ru/products-automatic/about) — быстрый способ настроить кампанию в Директе. Не нужно разбираться с настройками — достаточно указать ссылку на страницу сайта. Директ сам создаст, настроит и запустит кампанию с учетом особенностей бизнеса, подберет площадки, где больше целевой аудитории, направит бюджет на наиболее эффективные объявления и проконтролирует результаты кампании.
- [Мастер кампаний](ru/campaign-master/about) — инструмент для тех, кому нужно больше контроля за запуском кампаний. Мастер кампаний предложит тексты объявлений, изображения, поможет подобрать аудиторию и бюджет. Достаточно указать ссылку на продвигаемую страницу сайта или написать пару предложений о своем деле, если сайта нет.

Если вы измените Директ на Директ Про (расширенный интерфейс для профессиональных маркетологов), вернуться в Директ будет невозможно.

Подробнее об [интерфейсе Директ Про](ru/interface-direct-pro).

Внимание

Специалисты отдела клиентского сервиса могут вас проконсультировать только по кампаниям того логина, с которого вы обращаетесь. Логин можно увидеть, если открыть [ya.ru](http://ya.ru/) на соседней вкладке браузера. Специалист получит доступ к вашим данным только при обработке обращения.

Сканируйте QR-код или нажмите на него для перехода по ссылке.


При выборе Telegram, WhatsApp учитывайте, что Яндекс не контролирует, как сторонние мессенджеры хранят ваши данные и переписку на своей стороне, и не несет за это ответственность.
