## Технические требования

Для размещения медийной рекламы в формате премиум-билборда предоставьте файлы HTML5. Они должны соответствовать [требованиям к HTML-баннерам](ru/products-cpm-campaign/requirements). Все файлы, относящиеся к одной единице рекламы, должны иметь одно и то же имя.

Баннер не должен мимикрировать под оформление площадки — нельзя использовать фирменные цвета, элементы интерфейса. Креатив не должен сливаться с фоном сайта.

Запуск выбранного формата возможен только при наличии в кампании всех размеров баннеров, указанных в таблице. Например, запуск формата «Премиум-билборд. Десктоп» не состоится, если вы загрузили только один баннер размера 100% × 250. Для старта кампании необходимо также загрузить второй баннер размера 100% × 180.

|  |  |  |  |
| --- | --- | --- | --- |
|  | **Премиум-билборд. Десктоп** | **Премиум-билборд. Мобайл** | **Премиум-билборд. Кросс** |
| **Размер в пикселях** | 2 креатива: 100% × 250, 100% × 180 | 2 креатива: 100% × 120, 100% × 150 | 4 креатива: 100% × 250, 100% × 180, 100% × 120, 100% × 150 |
| **Максимальный вес** | 512 КБ | | |
| **Формат** | HTML5 | | |

Требования к десктопным баннерам

Требования к мобильным баннерам

1. Размер баннера может отличаться в зависимости от разрешения экрана и размера активного окна браузера пользователя.
2. Минимальный размер:

   - 984 × 250 пикселей для креатива 100% × 250;
   - 984 × 180 пикселей для креатива 100% × 180.
3. Баннер должен растягиваться на 100% ширины экрана, корректно отображаясь в диапазоне ширины экрана пользователя. При изменении размера баннера вся значимая информация должна быть отцентрована по ширине 984 пикселей.
4. Загружаемые HTML5-баннеры должны иметь размеры: один — 100% × 250 пикселей, другой — 100% × 180 пикселей. Размер баннера указывается в элементе `<meta>`:

   ```
   <meta name="ad.size" content="width=100%,height=250">
   ```

   ```
   <meta name="ad.size" content="width=100%,height=180">
   ```
5. Не принимаются баннеры с мигающими крупными графическими или текстовыми элементами и/или фоном, с резкими перемещениями элементов, существенно отвлекающие пользователя от взаимодействия со страницей сайта (изучения содержания страницы или ввода запроса), а также баннеры, заполненные текстом или изображениями низкого качества.
6. Рамка по периметру баннера не допускается. Исключение: если фон баннера или его нижняя часть белого цвета (`RGB #ffffff`), добавьте серую линию (`RGB #f3f3f3`) толщиной 1 пиксель по нижней грани баннера, чтобы рекламные материалы не сливались с шапкой сервиса.
7. В баннерах десктоп-формата (100% × 250 и 100% × 180) запрещено использование диплинк-ссылок. Ограничение связано с техническими особенностями обработки диплинк-ссылок на разных устройствах.

1. Размер баннера может отличаться в зависимости от разрешения экрана и размера активного окна браузера пользователя.
2. Минимальный размер:

   - 300 × 120 пикселей для креатива 100% × 120;
   - 300 × 150 пикселей для креатива 100% × 150.
3. Баннер должен растягиваться на 100% ширины экрана, корректно отображаясь в диапазоне ширины экрана пользователя. При изменении размера баннера вся значимая информация должна быть отцентрована по ширине 300 пикселей.
4. Загружаемые HTML5-баннеры должны иметь размеры: один — 100% × 120 пикселей, другой —100% × 150 пикселей. Размер баннера указывается в элементе `<meta>`:

   ```

   ```

   ```

   ```
5. Не принимаются баннеры с мигающими крупными графическими или текстовыми элементами и/или фоном, с резкими перемещениями элементов, существенно отвлекающие пользователя от взаимодействия со страницей сайта (изучения содержания страницы или ввода запроса), а также баннеры, заполненные текстом или изображениями низкого качества.
6. Рамка по периметру баннера не допускается. Исключение: если фон баннера или его нижняя часть белого цвета (`RGB #ffffff`), добавьте серую линию (`RGB #f3f3f3`) толщиной 1 пиксель по нижней грани баннера, чтобы рекламные материалы не сливались с шапкой сервиса.

Требования к оформлению баннера см. в разделе [Требования к HTML5-баннерам](ru/products-cpm-campaign/requirements).

## Требования модерации

Не принимаются баннеры следующих тематик:

- политика;
- магия и эзотерика;
- религиозные товары и услуги;
- услуги коллекторов;
- ритуальные товары и услуги;
- сайты знакомств или брачных агентств;
- социальная реклама, в том числе благотворительные организации;
- услуги детективов и частных охранных предприятий;
- товары и услуги для взрослых (например, секс-игрушки, эротические материалы и т. п.);
- аксессуары для оружия;
- конструктивно сходные с оружием изделия и аксессуары для них;
- военная тематика (перевозка военных, юридическая помощь в получении военного билета и т. п.);
- услуги по оптимизации сайтов и SEO;
- услуги по дезинфекции и дезинсекции;
- некоторые финансовые услуги и обучающие курсы по ним (микрофинансирование, предложение займов от частных лиц, сделки с бинарными опционами, деятельность форекс-дилеров, сделки с форекс-дилерами и т. п.);
- иные тематики, не соответствующие редакционной политике Яндекса.

Этот список не является полным и может быть изменен Яндексом в любой момент.
