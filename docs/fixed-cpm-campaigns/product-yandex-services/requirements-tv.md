## Технические требования

|  |  |
| --- | --- |
| Размер в пикселях | 1500 × 180 |
| Максимальный вес файла | 512 КБ |
| Формат | JPG, PNG |

Билборд не должен мимикрировать под оформление площадки — нельзя использовать фирменные цвета, элементы интерфейса. Креатив не должен сливаться с фоном сайта.

Для загрузки кампании в Директе можно использовать диплинк, собранный по инструкции ниже. Другие варианты диплинков не допускаются.

Как сформировать диплинк

1. Подготовьте параметр **[url]** — стандартный адрес для открытия в браузере, если реклама покажется не на ТВ.

   Пример: `https://hd.kinopoisk.ru/film/16498f8dea5d4e579bb1cae8fd026a0e`.
2. Подготовьте параметр **[store package]** — package приложения в сторе Яндекс ТВ для открытия страницы в сторе, если приложение еще не установлено.

   Пример: `ru.kinopoisk.yandex.tv`.
3. Подготовьте параметр **[url encoded deeplink]** — cтандартный диплинк на нужную страницу.

   Пример: `kpatv://film/16498f8dea5d4e579bb1cae8fd026a0e`.

   1. В конце полученной ссылки можно добавить utm-метки, если они поддерживаются в приложении.

      Пример utm-метки: `?utm_campaign=film_promo`.

      Ссылка с utm-меткой: `kpatv://film/16498f8dea5d4e579bb1cae8fd026a0e?utm_campaign=film_promo`.
   2. Преобразуйте полученную ссылку через URL-кодер (например, с помощью сервиса [URL Encoder](https://www.urlencoder.org/)).

      До преобразования: `kpatv://film/16498f8dea5d4e579bb1cae8fd026a0e?utm_campaign=test_yatv`.

      После преобразования: `kpatv%3A%2F%2Ffilm%2F16498f8dea5d4e579bb1cae8fd026a0e%3Futm_campaign%3Dtest_yatv`.
4. Если исходная ссылка **не содержит дополнительных параметров**, сформируйте из полученных параметров ссылку по шаблону: `[url]?package=[store package]&deeplink=[url encoded deeplink]`.

   Результат: `https://hd.kinopoisk.ru/film/16498f8dea5d4e579bb1cae8fd026a0e?package=ru.kinopoisk.yandex.tv&deeplink=kpatv%3A%2F%2Ffilm%2F16498f8dea5d4e579bb1cae8fd026a0e%3Futm_campaign%3Dtest_yatv`.
5. Если исходная ссылка **содержит дополнительные параметры** (например utm-метки), вместо `?` в ссылку добавьте `&` и сформируйте из полученных параметров ссылку по шаблону: `[url]&package=[store package]&deeplink=[url encoded deeplink]`.

   Результат: `https://hd.kinopoisk.ru/film/16498f8dea5d4e579bb1cae8fd026a0e?utm_campaign=film_promo&package=ru.kinopoisk.yandex.tv&deeplink=kpatv%3A%2F%2Ffilm%2F16498f8dea5d4e579bb1cae8fd026a0e%3Futm_campaign%3Dtest_yatv`.

Важно

На ограниченной доле устройств из-за особенностей версии прошивки диплинк может открываться в браузере.

## Расположение текста и элементов билборда

Обратите внимание, что иллюстрация представлена в уменьшенном размере.

1. Расстояние от края билборда до текста или кнопки должно быть не меньше 30 pt.
2. Текст и элементы билборда не должны располагаться в сейфзонах.

   Размер сейфзон:

   - метка — 60 × 20;
   - три точки — 42 × 42.

## Требования модерации

Для размещения не принимаются билборды следующих тематик:

- политика;
- магия и эзотерика;
- религиозные товары и услуги;
- услуги коллекторов;
- ритуальные товары и услуги;
- социальная реклама, в том числе благотворительные организации;
- услуги детективов и частных охранных предприятий;
- товары и услуги для взрослых (например, секс-игрушки, эротические материалы и т. п.);
- аксессуары для оружия;
- конструктивно сходные с оружием изделия и аксессуары для них;
- военная тематика (перевозка военных, юридическая помощь в получении военного билета и т. п.);
- услуги по оптимизации сайтов и SEO;
- услуги по дезинфекции и дезинсекции;
- некоторые финансовые услуги и обучающие курсы по ним (микрофинансирование, предложение займов от частных лиц, сделки с бинарными опционами, деятельность форекс-дилеров, сделки с форекс-дилерами и т. п.);
- иные тематики, не соответствующие редакционной политике Яндекса.
