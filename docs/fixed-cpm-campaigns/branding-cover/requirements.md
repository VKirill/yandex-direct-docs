Для размещения медийной рекламы в формате «Брендирование обложки dzen.ru» предоставьте файлы HTML5.

|  |  |  |  |
| --- | --- | --- | --- |
|  | **Брендирование обложки dzen.ru. Десктоп** | **Брендирование обложки dzen.ru. Мобайл** | **Брендирование обложки dzen.ru. Кросс** |
| **Размер в пикселях** | 100% × 300 | 100% × 301 | 2 креатива: 100% × 300 и 100% × 301 |

Запуск выбранного формата возможен только при наличии в кампании всех размеров баннеров, указанных в таблице. Например, запуск формата «Брендирование обложки dzen.ru. Кросс» не состоится, если вы загрузили только один баннер размера 100% × 300. Для старта кампании необходимо также загрузить второй баннер размера 100% × 301.

В зависимости от типа устройства некоторые технические требования будут различаться.

## Технические требования

1. Формат: только HTML5. Подробнее о [требованиях к HTML-баннерам](../../products-cpm-campaign/requirements.md). Все файлы, относящиеся к одной единице рекламы, должны иметь одно и то же имя.
2. Максимальный вес архива: 512 КБ.
3. Максимальный вес файла `index.html` в архиве: 150 КБ.
4. Баннер должен занимать 100% ширины экрана и корректно отображаться при любой ширине экрана пользователя. При изменении размера баннера вся значимая информация должна быть отцентрована:

   - для десктопного формата — по ширине 968 пикселей;
   - для мобильного формата — по ширине 320 пикселей.
5. Размер баннеров:

   Десктопные баннеры

   Мобильные баннеры

   100% × 300 пикселей. Размер баннера указывается в элементе `<meta>`:

   ```
   <meta name="ad.size" content="width=100%,height=300">
   ```

   100% × 301 пикселей. Размер баннера указывается в элементе `<meta>`:

   ```
   <meta name="ad.size" content="width=100%,height=301">
   ```
6. В секции `<head>` баннер должен содержать метатег `<meta>` с атрибутом `ad.bgrcolor`. Атрибут `content` задает цвет подложки в формате HEX, состоящий из шестизначной комбинации цифр и букв, перед которой может стоять знак «#».

   Примеры:

   ```
   <meta name="ad.bgrcolor" content="FFFAAA">
   <meta name="ad.bgrcolor" content="#FFFAAA">
   ```
7. Все обработчики событий должны быть явно указаны внутри тега `<script>`, например:

   ```
   <script>
   document.onload = init;
   </script>
   ```
8. Рамка по периметру баннера не допускается.
9. В баннере должны быть соблюдены обязательные сейфзоны:

   Десктопные баннеры

   Мобильные баннеры

   - **Верхняя часть** — зона поисковой строки Яндекса (цветная фоновая часть баннера без элементов креатива). Занимает 110 пикселей, из них 10 пикселей до поисковой строки с виджетами и 16 пикселей — после;
   - **Центральная часть** — зона с основными элементами креатива. Занимает 150 пикселей.
   - **Нижняя часть** — цветная фоновая часть баннера без элементов креатива, которая располагается перед контентом страницы. Занимает 40 пикселей.
   - **Зона бейджа «Реклама»**. Располагается вертикально справа внизу. В зоне бейджа «Реклама» текст или важные элементы баннера не размещаются.
     - Размеры зоны бейджа: 30 × 100 пикселей.

## Рекомендации по оформлению

#### Элементы креатива

- Не принимаются баннеры с эффектом мерцания, мигающими крупными графическими или текстовыми элементами или фоном.
- Баннеры не должны быть заполнены текстом или изображениями низкого качества.
- Анимация должна быть плавной, без резких переходов, чтобы не отвлекать пользователя от взаимодействия со страницей сайта (изучения содержания страницы или ввода запроса).
- Кнопка призыва к действию не должна занимать более 25% площади баннера.
- В баннере нельзя использовать мигающие кнопки, а также кнопки с надписями «SALE», «Распродажа» и т.д.
- Баннер не должен создавать впечатление каталога или карточки товара.

#### Рекомендованные цвета

- Баннер не должен мимикрировать под оформление площадки — нельзя использовать фирменные цвета, элементы интерфейса. Креатив не должен сливаться с фоном сайта. Вместо белого цвета следует использовать светло-серый, а вместо черного — темно-серый.
- Запрещено использовать более 25% площади баннера для красных, желтых и других агрессивных цветов или цветовых схем.
- Недопустимые фоны: яркие кислотные и неоновые цвета, а также сильный градиент, созданный с помощью разных цветов.

#### Оформление предупреждений и обязательной юридической информации

Вы можете указать предупреждения и обязательную юридическую информацию в нижней части баннера вместе с основным объектом рекламы. Например, сменяющимися слайдами.

Требования к шрифту и отображению текста:

- Минимальный размер шрифта — 22 пикселя.
- Текст должен быть контрастным и легко читаемым.
- Информация должна отображаться на экране не менее трех секунд.
- Информация не должна занимать весь кадр.

## Требования модерации

Не принимаются баннеры следующих тематик:

- политика;
- магия и эзотерика;
- религиозные товары и услуги;
- услуги коллекторов;
- ритуальные товары и услуги;
- сайты знакомств или брачных агентств;
- социальная реклама, в том числе благотворительные организации;
- услуги детективов и частных охранных предприятий;
- товары и услуги для взрослых (например, секс-игрушки, эротические материалы и т. п.);
- аксессуары для оружия;
- конструктивно сходные с оружием изделия и аксессуары для них;
- военная тематика (перевозка военных, юридическая помощь в получении военного билета и т. п.);
- услуги по оптимизации сайтов и SEO;
- услуги по дезинфекции и дезинсекции;
- некоторые финансовые услуги и обучающие курсы по ним (микрофинансирование, предложение займов от частных лиц, сделки с бинарными опционами, деятельность форекс-дилеров, сделки с форекс-дилерами и т. п.);
- иные тематики, не соответствующие редакционной политике Яндекса.
