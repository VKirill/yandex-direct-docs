# Конструктор креативов

- [Создание креатива](../products-image-ads/builder.md#section_qhm_jrm_b3b)
- [Добавление креативов из библиотеки](../products-image-ads/builder.md#library)
- [Экспорт креативов](../products-image-ads/builder.md#export)

Конструктор креативов помогает вам самостоятельно создать привлекательные графические объявления в Единой перфоманс-кампании. Адаптивный креатив автоматически подстраивается под блок для размещения объявлений и дает возможность показывать объявление на максимальном количестве площадок.

Вы можете создавать неограниченное количество креативов бесплатно.

Для создания не нужны специальные знания или опыт. Всего за несколько минут вы получите объявление без помощи профессионального дизайнера.


## Создание креатива

1. При создании или редактировании графического объявления в Единой перфоманс-кампании в блоке **Креатив** нажмите **Добавить** → **Создать новый креатив**. Откроется окно конструктора креативов.
2. В правой части окна выберите изображение. Вы можете добавить свое, выбрать готовое из библиотеки или ранее загруженное изображение. Все изменения настроек сразу отображаются в области предварительного просмотра.
3. Установите смарт-центр — главную по смыслу область, которая не должна быть потеряна при адаптации объявления под блок. Изменить смарт-центр можно в блоке **Изображение**, нажав кнопку **Изменить**.
4. Выберите цвет заливки и рамки, добавьте основной текст. Цвет рамки не должен совпадать с фоном (цветом заливки и цветами загруженного файла).
5. Оформите кнопку — выберите цвет и добавьте текст. Добавьте логотип и домен, в поле **Дополнительный текст** добавьте специальное предложение. Информация будет отображаться в объявлении, если позволит формат блока.
6. В креативе должна присутствовать вся информация, необходимая согласно требованиям законодательства и [правилам Яндекса](../moderation/ad-rules.md#image-ads), в том числе [предупреждения и возрастные ограничения](../moderation/age-alerts.md#image-ads). Укажите нужные данные в блоках **Возрастное ограничение**, **Предупреждение** и **Юридическая информация**.
7. После того как вы закончите работу над креативом, нажмите кнопку **Создать**. Креатив будет добавлен в библиотеку. Используйте его для создания объявления.

## Добавление креативов из библиотеки

Вы можете использовать созданные креативы в других группах.

Ограничение

В графических объявлениях в Единой перфоманс-кампании креативы из конструктора можно заменить только на креативы из конструктора того же размера либо адаптивный, но не на креативы, загруженные с компьютера или из интернета.

## Экспорт креативов

Ссылки на все креативы, которые были созданы в конструкторе, можно выгрузить в файл формата XLS. Для этого нажмите кнопку **Экспортировать список в XLS**.

Ссылки можно использовать, чтобы добавить креативы сразу в несколько объявлений с помощью [XLS/XLSX-файлов](../alternative-interfaces/xls-interface.md).

Ограничение

Перенести креативы с одного аккаунта на другой нельзя.
