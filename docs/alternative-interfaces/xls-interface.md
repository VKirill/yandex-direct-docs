Управление кампаниями с помощью файлов формата XLS и XLSX

# Управление кампаниями с помощью файлов формата XLS и XLSX

- [Баллы и ограничения](ru/alternative-interfaces/xls-interface#limits)
- [Как работать с XLS/XLSX-файлами](ru/alternative-interfaces/xls-interface#how-to)
- [Заполнение полей файла](ru/alternative-interfaces/xls-interface#fields)
  - [Как узнать ID видео](ru/alternative-interfaces/xls-interface#video-id)

Управлять кампаниями в Директе можно не только в интерфейсе, но и с помощью файлов формата XLS и XLSX. Эта функция позволяет существенно сэкономить ваше время при работе с большим количеством объявлений, а также работать с кампаниями и объявлениями, не имея доступа к интернету.

Например, вы выгружаете файл с данными кампании, вносите изменения в режиме офлайн, а затем загружаете обновленный файл обратно. После этого объявления в интерфейсе автоматически обновятся в соответствии с изменениями в файле. Вы можете редактировать существующие кампании и создавать новые, копировать группы объявлений или ключевые фразы из одной кампании в другую.


Внимание

При копировании кампании с помощью файлов формата XLS и XLSX не сохраняются настройки стратегии. По умолчанию в новой кампании будет установлена стратегия **Максимум кликов** без ограничения бюджета с показами на всех площадках. Не забудьте изменить настройки стратегии после копирования.

## Баллы и ограничения

Возможность управления кампаниями с помощью XLS/XLSX-файлов предоставляется бесплатно, но в связи с техническими особенностями Директа имеет некоторые ограничения. Количество и виды операций, которые доступны в течение суток, зависят от количества баллов на аккаунте. Баланс пересчитывается один раз в сутки в ночное время. Все баллы, не потраченные за прошедшие сутки, сгорают.

На количество доступных для использования баллов влияет множество факторов, например количество отклоненных на модерации объявлений, количество ключевых фраз с низким CTR, средний CTR кампании, средний бюджет кампании.

Баллы списываются при следующих операциях:

|  |  |
| --- | --- |
| Добавление нового объявления | 12 баллов |
| Редактирование объявления | 4 балла |
| Добавление новой ключевой фразы | 2 балла |
| Редактирование ключевой фразы | 1 балл |

Количество потраченных за одну загрузку баллов отображается на странице подтверждения загрузки. Баллы списываются только при успешном импорте данных. Если при загрузке возникают ошибки, баллы не списываются.

Внимание

Яндекс оставляет за собой право приостановить доступ к управлению кампаниями с помощью XLS/XLSX‑файлов для отдельных пользователей без объяснения причин.

## Как работать с XLS/XLSX-файлами

С помощью XLS/XLSX-файлов можно редактировать существующие кампании и создавать новые. Чтобы отредактировать существующую кампанию, необходимо сначала выгрузить ее в файл, внести изменения, а затем загрузить обновленный файл обратно. Для создания новой кампании используйте [шаблон файла](https://yastatic.net/s3/doc-binary/src/support/direct/ru/direct_example2.xls).

В меню выберите **Инструменты** → **Управление с помощью Excel** и в открывшемся окне переключитесь на нужную вкладку.

Выгрузка

Загрузка

История

Выберите в выпадающем списке кампанию, в которую хотите внести изменения, и укажите формат файла (XLS или XLSX).

Выберите файл, который хотите загрузить. Он должен быть формата XLS или XLSX, размером не более 60 000 строк. Структура должна совпадать с [шаблоном](https://yastatic.net/s3/doc-binary/src/support/direct/ru/direct_example2.xls). Если вы хотите указать общий регион для всех объявлений, установите соответствующий флажок.


Выберите одно из действий:

- **изменить кампанию** — объявления будут добавлены в кампанию, номер которой указан в поле **№ заказа** в XLS/XLSX-файле. Если поле **№ заказа** не заполнено или заполнено неверно, то действие будет недоступно;
- **создать новую кампанию** — объявления будут добавлены в новую кампанию, в ней будет создано столько же объявлений, сколько в файле содержится строк с различными значениями полей **Заголовок**, **Текст** или **Ссылка**;
- **добавить в существующую кампанию** — объявления будут добавлены в кампанию, которую вы выберете в выпадающем списке. Номер, который указан в поле **№ заказа**, не учитывается. Если вы загружаете уже созданные объявления из одной кампании в другую, система воспримет их как новые и создаст дубликаты.

Если после загрузки файла обнаружены ошибки, то данные не будут загружены и будет показано сообщение об ошибке. [Подробнее о возможных ошибках](ru/alternative-interfaces/xls-interface-errors).

Если в файле нет ошибок, вы можете продолжить загрузку. Кампания автоматически блокируется от изменений на 5 минут. Для продолжения загрузки согласитесь с условиями оферты.

Просмотрите списки загруженных и выгруженных файлов с указанием даты операции и номера кампании.

Если при работе с файлом произошла ошибка загрузки изображения (например, размер, формат файла или ссылка на него не соответствуют требованиям), в строке вы увидите соответствующий статус. Там же вы найдете ссылку на скачивание отчета об ошибках.

## Заполнение полей файла

Файл состоит из нескольких вкладок:

- **Тексты** — основная вкладка для внесения изменений, на ней находятся данные рекламной кампании и все параметры объявлений. Некоторые поля этой вкладки не подлежат редактированию.
- **Регионы** — справочник значений поля **Регион**. Данные носят информационный характер, вносить изменения на вкладке не нужно.
- **Словарь значений полей** — справочник значений полей **Тип кампании** и **Тип объявления**. Данные носят информационный характер, вносить изменения на вкладке не нужно.

Набор полей вкладки **Тексты** зависит от типа кампании. Описание заполнения полей приведено в таблице.

Примечание

Опция доступна для Единой перфоманс-кампании — только для файлов с комбинаторными, текстово‑графическими и графическими объявлениями.

### Как узнать ID видео

На странице редактирования объявления в блоке **Видео** ID видео указано под роликом.
