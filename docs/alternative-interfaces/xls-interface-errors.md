Ошибки при управлении кампанией с помощью файлов формата XLS и XLSX

# Ошибки при управлении кампанией с помощью файлов формата XLS и XLSX

|  |  |
| --- | --- |
| **Ошибка**\* | **Решение** |
| Строки %s: недопустимые символы. | Вы неправильно заполнили поле **Метки** в указанной строке.  В поле **Метки** укажите [метки объявлений](ru/campaigns/tags), а не UTM-метки. UTM-метки добавляйте в поле **Ссылка**. |
| Строка %s: не указано значение в поле Phrase. | Вы не заполнили поле **Фраза (с минус-словами)** в указанной строке.  Заполните поле **Фраза (с минус-словами)** для главного объявления группы. |
| Не удалось распознать названия колонок. | Вы изменили названия столбцов в исходном файле. Проверьте, что названия всех столбцов совпадают с названиями столбцов в [шаблоне](https://direct.yandex.ru/public/direct_example.xls?cmd=exportSampleCamp&new_format=1). |
| Строка %s: текст объявления написан на украинском языке и не соответствует установленному региону показа. Возможен таргетинг только на Украину. | Вы заполнили поля **Заголовок** и **Текст** на украинском языке в указанной строке.  Язык объявления должен соответствовать региону показа. Измените регион на Украину или замените текст на русский. |
| Строка %s: группа не может состоять только из доп. объявлений. | Вы неправильно заполнили поле **Доп. объявление группы** в указанной строке.  В поле **Доп. объявление группы** главное объявление отмечается знаком «`-`», остальные объявления группы отмечаются в этом поле знаком «`+`». В группе может быть только одно главное объявление. |
| Строка %s: доп. объявление группы должно иметь значение «+» или «-» вместо %s. | Вы неправильно заполнили поле **Доп. объявление группы** в указанной строке.  В поле **Доп. объявление группы** главное объявление отмечается знаком «`-`», остальные объявления группы отмечаются в этом поле знаком «`+`». Другие символы не допускаются. |
| Группа объявлений %s не существует. | Вы неправильно заполнили поле **ID группы** в указанной группе.  При внесении изменений в существующую кампанию не допускается изменение поля **ID группы**. При создании новой кампании или группы объявлений поле **ID группы** должно оставаться пустым. |
| Объявление %s не существует. | Вы неправильно заполнили поле **ID объявления** в указанном объявлении.  При внесении изменений в существующую кампанию не допускается изменение поля **ID объявления**. При создании новой кампании или объявления поле **ID объявления** должно оставаться пустым. |
| Контактная информация: не указано значение в поле «Время работы». | Вы неправильно заполнили поле **Время работы** на вкладке **Контактная информация**.  Убедитесь, что формат ячейки текстовый и время указано в формате ЧЧ:ММ, например 18:00. |
| Значения в поле %s отличаются в строках %s. Данные значения будут объединены. | В указанных строках значения поля **ID объявления** совпадают. Система считает все строки с одинаковым **ID объявления** одним объявлением. Все строки будут объединены в одно объявление.  Если вы не хотите объединять объявления, удалите значения поля **ID объявления**. Обратите внимание, если вы вносите изменения в уже существующие объявления, значения поля **ID объявления** нужно оставить как в исходном файле. |
| Значения в поле %s отличаются в строках %s. Указанные поля должны быть одинаковыми для одной и той же группы. | Вы неправильно заполнили указанное поле. Значения в поле должны совпадать для всех объявлений группы.  Система считает все строки файла с одинаковым **ID группы** одной группой объявлений. У одной группы объявлений **Название группы** и **Номер группы** должны совпадать.  Если вы не хотите добавлять объявление в уже существующую группу, то оставьте поле **ID группы** пустым. |
| Фраза, регион, ставка и метки не указываются в дополнительных объявлениях группы. Эти поля в строке %s будут проигнорированы. | Вы заполнили лишние поля в указанной строке. Эти данные не будут внесены.  Поля **Фраза (с минус-словами)**, **Регион**, **Ставка**, **Ставка в сетях** и **Метки** заполняются только для главного объявления группы. В главном объявлении указываются параметры, общие для всех объявлений группы. В поле **Доп. объявление группы** оно отмечается знаком «`-`». |

\* Вместо %s будет подставлен номер строки, номер объявления или название поля.

*Ознакомьтесь с нашими рекомендациями по управлению кампаниями с помощью файлов формата XLS и XLSX в [Справке](ru/alternative-interfaces/xls-interface)*
