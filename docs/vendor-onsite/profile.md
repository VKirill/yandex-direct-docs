# Заполните профиль и приложите документы

- [Заполните данные рекламодателя](ru/vendor-onsite/profile#zapolnite-dannye-reklamodatelya)
- [Заполните данные вендора](ru/vendor-onsite/profile#zapolnite-dannye-vendora)

На странице **Настройки** заполните профиль.

## Заполните данные рекламодателя

Без данных рекламодателя создать кампанию не получится.

1. Выберите вид организации.
2. Внесите наименование и ИНН вашей организации. Если вы — физическое лицо, укажите свои фамилию, имя, отчество.
3. Заполните данные плательщиков.
4. Нажмите кнопку **Сохранить**.

## Заполните данные вендора

На основе данных вендора будет формироваться список доступных вам магазинов.

**Блок Вендор**

1. Введите фирменное название нового вендора. После одобрения заявки продолжите заполнение профиля. Если вендор уже был ранее добавлен, выберите из списка нужное название.
2. В поле **Варианты написания вендора** укажите все возможные варианты названия вендора.

   У разных ритейлеров один и тот же вендор может называться по-разному (например, название может быть написано с помощью латиницы, кириллицы или содержать специальные символы). Алгоритмы Директа сопоставят загруженные фиды ритейлеров с данными из профиля вендора и предложат вам больше магазинов. Это позволит получить больше конверсий.

**Блок Бренды**

1. Выберите роль — представитель вендора или агентство.
2. Укажите ваш бренд и добавьте все возможные варианты его написания.
3. Загрузите документы, которые подтверждают права владения товарным знаком.

   Если бренд зарегистрирован на указанное в кабинете юридическое лицо

   Загрузите один из следующих документов:

   - Свидетельство о регистрации товарного знака в Роспатенте.
   - Электронная выписка из Всемирной организации интеллектуальной собственности (WIPO) или Федерального института промышленной собственности (ФИПС).

   Если бренд зарегистрирован на другое лицо

   Загрузите один из следующих документов:

   - Лицензия на использование товарного знака.
   - Иной документ от правообладателя, подтверждающий право на использование товарного знака, например письмо-согласие, адресованное лицу, указанному в кабинете.
4. Если у вас несколько брендов, нажмите **Добавить ещё** и повторите пункты 1–3.

Нажмите кнопку **Сохранить**, чтобы отправить данные вендора на модерацию. После успешного прохождения проверки можно приступить к созданию кампании.

#### Остались вопросы?

Внимание

Специалисты отдела клиентского сервиса могут вас проконсультировать только по кампаниям того логина, с которого вы обращаетесь. Логин можно увидеть, если открыть [ya.ru](http://ya.ru/) на соседней вкладке браузера. Специалист получит доступ к вашим данным только при обработке обращения.

Сканируйте QR-код или нажмите на него для перехода по ссылке.


При выборе Telegram, WhatsApp учитывайте, что Яндекс не контролирует, как сторонние мессенджеры хранят ваши данные и переписку на своей стороне, и не несет за это ответственность.
