# Продвижение товаров в интернет-магазинах

- [Создание кампании](ru/vendor-onsite/performance-campaign#sozdanie-kampanii)
- [Статистика](ru/vendor-onsite/performance-campaign#statistika)

Продвижение товаров в интернет-магазинах — это тип кампании в Директе для вендоров, который позволяет показывать click-in-объявления с карточками товаров на сайтах и в приложениях интернет-магазинов. Кампания работает по стратегии увеличения количества продаж, при этом стоимость клика не будет превышать заданную вами цену.

## Создание кампании

1. В меню слева нажмите **Кампания** или на странице **Кампании** нажмите **Добавить кампанию** и выберите **Перфоманс-кампания click-in**.
2. На основе фидов ритейлеров и данных из профиля вендора система предложит список доступных вам магазинов (сайтов и приложений ритейлеров) сети [Ритейл Медиа](https://retailmedia.yandex.ru/promo). В блоке **Магазин** выберите один из них.

   Посмотреть
3. Укажите недельный бюджет и цену клика.
4. Задайте срок проведения кампании.
5. В блоке **Товары** добавьте бренд и категории товаров, которые хотите продвигать, или отдельные товары. Для каждого товара будет создано отдельное объявление.
6. Настройте корректировки:

   - [Пол и возраст](ru/impressions/bids-adjustment#socdem-bids-adjusment)
   - [Устройства](ru/impressions/bids-adjustment#devices)
   - [Платежеспособность](ru/impressions/bids-adjustment#solvency)
   - [Регион показа](ru/impressions/bids-adjustment#region)
7. Нажмите **Запустить**.

## Статистика

После запуска кампании в Мастере отчетов будет доступна статистика: показы, клики, конверсии и другие метрики по каждому объявлению. Чтобы посмотреть статистику кампании, в меню слева нажмите **Статистика** и [настройте](ru/statistics/about#settings) нужные отчеты.
