# Статистика кампании

Статистика по вашему размещению будет доступна в [Мастере отчетов](../statistics/about.md).

Чтобы оценить работу кампании, добавьте метрики для наружной рекламы:

1. Нажмите **Группировки и метрики**.
2. Выберите **Метрики** → **Охватная реклама** → **Наружная реклама (DOOH)**.
3. Нажмите **Применить**.

Чтобы оценить статистику в разрезе экранов:

1. Нажмите **Группировки и метрики**.
2. Выберите **Группировки** → **Площадки** → **Название площадки**.
3. Нажмите **Применить**.

Для удобства вы можете [сохранить отчет](../statistics/about.md#save-edit).
