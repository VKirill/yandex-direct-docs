# Представителям производителей бытовой техники и электроники

Яндекс может отказать в продвижении сервисных центров с указанием на официальность, авторизацию или сертификацию для вашего бренда, если их нет в списках ваших официальных сервисных центров. Для проверки объявлений о сервисных центрах вашего бренда заполните форму ниже.

Чтобы создать правило для проверки на официальность, передайте нам **список сайтов ваших авторизованных сервисов** одним из способов:

- в файле любого формата;
- или разместите на сайте и пришлите ссылку на страницу.

Если у вас нет списка сайтов, мы не сможем поддержать это правило для вашего бренда.
