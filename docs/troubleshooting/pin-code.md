# PIN-код

При звонке в [отдел клиентского сервиса Яндекс Директа](../contact-us.md) специалисты запрашивают номер кампании и разовый PIN-код. Он действует в течение 15 минут и требуется для каждого нового обращения по телефону. Это нужно для безопасности данных: только вы можете выдать доступ специалисту к вашим кампаниям и только на время звонка.

Для создания PIN-кода в правом верхнем углу страницы нажмите на логин и выберите **Получить PIN-код**.
