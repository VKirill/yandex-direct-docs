# Вопросы и ответы

### Не формируются объявления для товаров

Внимание

Убедитесь, что на счете есть деньги. Генерация объявлений не начнется, если нет средств.

Не формируются объявления по фиду (по ссылке или с компьютера)

Проверьте, что выполнены условия:

- Фид соответствует [требованиям](../feeds/requirements.md).
- В фиде есть офферы, которые соответствуют условию фильтра:

  - Если в фильтре выбран параметр **Только в наличии**, в фиде должны присутствовать товары с атрибутом `available = "true"`. Если в условии есть другие фильтры, например по товарам определенной категории, то в наличии должны быть товары именно этой категории.
  - При настройке фильтра по производителям в фиде должны быть товары с элементом `vendor`. Значение элемента должно соответствовать значению, указанному в фильтре.
  - Если задан фильтр по элементу `categoryId`, в фиде должны быть товары указанной категории. Если в категории нет товаров, то товарные объявления этой категории не сформируются. Если в родительской категории (категория имеет вложенные категории с атрибутом `parentId`) есть подкатегории, которые содержат товары, фильтр, настроенный на родительскую категорию, в которой товаров нет, работать не будет.

Не формируются объявления для товаров, найденных на сайте

Проверьте, что выполнены условия:

- Сайт соответствует [требованиям](../unified-performance-campaign/structure-of-site.md).
- Сайт доступен.
- На сайте есть товары, которые соответствуют условиям фильтров:

  - **Заголовок** соответствует заголовку товара на сайте.
  - **Разделы сайта** соответствуют ссылкам на страницы с товарами.

  - **Ссылка на товар** соответствует ссылке товара на сайте.
  - **Цена товара** соответствует цене на сайте.

Не формируются объявления, созданные с помощью ручного управления товарами

Проверьте, что товары в карточке соответствуют условиям фильтра:

- Если в фильтре выбран параметр **Только в наличии**, проверьте, что товар в наличии на сайте.
- При настройке фильтра по параметру **Производитель** убедитесь, что значение элемента есть в карточке товара и соответствует фильтру.
- Если задан фильтр по параметру **Категория**, убедитесь, что значение элемента есть в карточке товара и соответствует фильтру.

### Не формируются объявления для страниц каталога

Внимание

Убедитесь, что на счете есть деньги. Генерация объявлений не начнется, если нет средств.

Не формируются объявления по фиду (файл с компьютера)

Объявления для страниц каталога могли не сформироваться, если не были соблюдены [требования к CSV-файлу](../product-campaign/create.md#catalog).

Не формируются объявления страниц каталога, найденных на сайте
