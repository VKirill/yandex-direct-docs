# Приостановка показов по отдельной ключевой фразе

Вы можете приостановить на время показ группы объявлений по определенной ключевой фразе. При этом накопленная по этой фразе статистика будет сохранена.

Остановить показ по выбранной ключевой фразе можно на странице кампании в блоке редактирования фраз.

Останавливать можно любые активные фразы. После остановки фраза будет подсвечена бледно-синим цветом.

Внимание

Cтатистика (накопленный CTR) по остановленным фразам будет сохраняться в течение 28 календарных дней с момента остановки, после чего обнулится. Однако фразу все равно можно будет включить в работу по прошествии этого времени.
