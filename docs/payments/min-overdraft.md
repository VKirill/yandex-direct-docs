# Минимальный ежемесячный расход для отсрочки платежа

|  |  |  |
| --- | --- | --- |
| **Валюта платежа или указанная в Клиентском веб-интерфейсе расчетная величина** | **Минимальный среднемесячный расход для предоставления отсрочки платежа** | **Минимальный период регулярного размещения объявлений** |
