# Розыск платежа

Чтобы проверить поступление денег или найти нужный номер счета для оплаты, на странице **Мои кампании** перейдите по ссылке **Платежи и документы** в раздел **Счета**.

Нажмите кнопку **Показать счета**.

Напротив оплаченных счетов в таблице будут указаны дата, сумма оплаты и номер платежного поручения, если платили через банк.

Если платеж не зачислен на нужный счет, вы можете отправить заявку на розыск платежа с помощью формы ниже. В форме укажите номер счета вместе с [буквенным префиксом](https://yandex.ru/support/balance/entities/bill.html) — он зависит от схемы оплаты и страны размещения. Например, для России это префикс «Б-».

Важно
