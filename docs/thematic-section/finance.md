Тематические разделы могут содержать информацию о банковских услугах: курсе валют, условиях ипотеки, вкладов и кредитов, дебетовых и кредитных картах, а также отзывы на указанные услуги и на банки. Это помогает пользователю сделать выбор, а банку — привлечь внимание выгодными услугами и условиями сделок.

Продвижение в тематических разделах продуктов категории Финансы доступно для банков и партнеров, входящих в реестр операторов финансовых платформ (далее — ОФП) ЦБ РФ.

Принимаются предложения партнеров, предоставляющие возможность оформления банковских продуктов на ресурсе ОФП, указанного в реестре ЦБ РФ.

Продвижение доступно в тематических разделах Кредитные карты и Кредиты наличными. Они показываются в ответ на релевантные запросы пользователей в тематических блоках выдачи поиска и в Теме Поиска «Финансы».

Пример отображения (дизайн может меняться):

|  |  |
| --- | --- |
| **Кредиты** | **Кредитные карты** |
|  |  |

Предложения в тематическом разделе ранжируются с учетом ставки (на которую влияет заданная цена целевого действия), показателя кликабельности и коэффициента релевантности. Объявления могут показываться на всех типах устройств.

## Как пользователь увидит персональные предложения

1. После запроса в тематическом разделе в поиске или Теме Поиска «Финансы» пользователь нажимает кнопку **Рассчитать**.
2. Авторизуется в Яндекс ID или подтверждает профиль. Затем происходит идентификация и анализ профиля пользователя на стороне банка.
3. Получает объявления с персонализированными предложениями от разных банков и кнопками с призывом к действию.

## Как настроить

1. Продвижение в тематических разделахβ работает при наличии API-интеграции с вашим сервисом. Настройка интеграции позволит:

   - Анализировать профиль авторизовавшегося пользователя и определять, подходит ли ему ваш продукт, а также показывать сформированные специально для него предложения.
   - Автоматически формировать и показывать предложения с учетом характеристик пользовательских профилей.
   - Получать заявки, которые пользователи оставят в тематических разделах.

   Инструкции по настройке API-интеграции:

   - [кредиты](../thematic-section/credits/about.md) — API для партнеров
   - [кредитные карты](../thematic-section/credit-cards/about.md) — API для партнеров
2. После настройки интеграции [настройте и запустите](../thematic-section/finance.md#create) кампанию в Директе.

### Создание и настройка кампании в Директе

1. Нажмите **Добавить** → **Кампанию**. На вкладке **Режим эксперта** выберите **Продвижение в тематических разделахβ**. Затем — **Финансы и страхование**.
2. В блоке **Продвигаемый продукт** выберите продукт, который вы будете продвигать: Кредитные карты или Кредит наличными. В одной кампании можно продвигать только один продукт.
3. В блоке **Бюджет и цена целевого действия** задайте следующие параметры:

   - **Бюджет** — ограничение расхода в неделю.
   - **Цена целевого действия** — это плата за клик по кнопке «Оформить». Цена не должна быть ниже установленного минимального порога — иначе показ объявления прекратится. Подсказка о минимальном пороге отображается в интерфейсе, если указанная цена ниже его значения. Об изменении минимального порога менеджер сообщит заранее.
4. В блоке **География показов** выберите подходящие регионы — объявления будут показаны только той аудитории, которая сможет воспользоваться вашим предложением, так как находится в выбранном регионе показа или интересуется им. Вы можете:

   - **Перечислить страны, области, города** — в свободной форме начните вводить нужный регион, например, Тула или Курск. Или воспользуйтесь деревом регионов.
   - **Указать регион на карте** — если вас интересуют клиенты в узкой области.

   [Подробнее о настройке](../efficiency/geotargeting.md).
5. Заполните параметры для API-интеграции:

   - **API** — укажите ссылку на API;
   - **Client ID** — ID пользователя для авторизации;
   - **Client Secret** — ключ для авторизации.

   Валидация API может занимать до 72 часов.
6. Запустите кампанию.

   После запуска данные интеграции уйдут на проверку. Как только проверка будет закончена, статус кампании изменится на «Активна».

## Оплата продвижения

Оплата только за целевое действие — клик по кнопке «Оформить».

Цена целевого действия списывается фиксированная, с учетом установленного недельного бюджета.

## Статистика

Результаты продвижения доступны в Мастере отчeтов и на странице **Кампании**.
