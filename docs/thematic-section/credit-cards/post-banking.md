## Получить все предложения банков

Получает актуальные предложения от банков-партнеров на основании номера телефона клиента. Является предпочтительным унифицированным каналом взаимодействия с источниками данных. Может быть использован для одобрения как кредитов, так и кредитных карт.

## Формат запроса

```
POST /api/banking/loan_approval
```

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Параметр** | **Описание** | **Тип** | **Обязательно** | **Комментарий** |
| `phone_number` | Номер телефона. | `string` | Да |  |
| `amount` | Кредитный лимит. | `number` | Нет |  |
| `product_type` | Наименование кредитного продукта. | `string` | Да | Значение из справочника:   - `"loan"` — кредит наличными; - `"credit_card"` — кредитная карта. |
| `search_id` | `ID` запроса. | `string` | Да | Параметр для повторных запросов, `UUID`. |
| `props` | Дополнительные служебные параметры. | `array` | Нет | Массив строк, который может содержать любую служебную информацию, оговоренную вне данной страницы. |

В случае асинхронной модели обработки данных:

Предполагается множественный опрос по `GET`. Выполняется, пока `status` в ответе не станет `done`.

```
GET /api/banking/loan_approval?search_id={{search_id}}
```

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Параметр** | **Описание** | **Тип** | **Обязательно** | **Комментарий** |
| `search_id` | `ID` запроса. | `string` | Да | Параметр для повторных запросов, `UUID`. |
