# Стандартные коды ответов

200 ОК

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **Параметр** | **Описание** | **Тип** | **Обязательно** | **Комментарий** |
| `search_id` | `ID` запроса. | `string` | Да |  |
| `status` | Статус запроса. | `string` | Да | Значение из справочника:   - `"pending"` — обрабатывается; - `"done"` — завершено. |
| `name` | Наименование организации. | `string` | Да |  |
| `decision` | Решение по заявке. | `string` | Да | Значение из справочника:   - `"has_approve"` — есть одобрение; - `"rejected"` — отклонено. |
| `data` | Массив с решениями. | `array` | Нет | Если предложение одно, то массив будет содержать один элемент. |
| `product_type` | Наименование кредитного продукта. | `string` | Да | Значение из справочника:   - `"loan"` — кредит наличными; - `"credit_card"` — кредитная карта. |
| `decision` | Решение по предложению. | `string` | Да | Значение из справочника:   - `"pre_approved"` — предодобрено; - `"full_approved"` — полностью одобрено. |
| `amount` | Сумма займа. | `object` | Да | Если продукт `"credit_card"`, то речь идет о кредитном лимите. |
| `amount.min` | Минимальная сумма займа. | `number` | Да |  |
| `amount.max` | Максимальная сумма займа. | `number` | Да |  |
| `term` | Срок займа. | `object` | Да, если `product_type = "loan"`. |  |
| `term.min` | Минимальный срок займа. | `number` | Да | Указывается в месяцах. |
| `term.max` | Максимальный срок займа. | `number` | Да | Указывается в месяцах. |
| `rate` | Ставка займа. | `object` | Да |  |
| `rate.min` | Минимальная cтавка. | `number` | Да |  |
| `rate.max` | Максимальная cтавка. | `number` | Да |  |
| `total_cost` | Полная стоимость кредита (ПСК). | `object` | Да |  |
| `total_cost.min` | Минимальная ПСК. | `number` | Да |  |
| `total_cost.max` | Максимальная ПСК. | `number` | Да |  |
| `grace_period` | Льготный период. | `number` | Да, если `product_type = "credit_card"`. | Указывается в днях. |
| `payment` | Ежемесячный платеж. | `object` | Да |  |
| `payment.min` | Минимальный ежемесячный платеж. | `number` | Да, если `product_type = "loan"`. |  |
| `payment.max` | Максимальный ежемесячный платеж. | `number` | Да, если `product_type = "loan"`. |  |
| `payment.cc_rate` | Минимальный платеж по кредитной карте. | `number` | Да, если `product_type = "credit_card"`. | Указывается в процентах: `payment.cc_rate = 10 // 10% от суммы долга`. |
| `special_conditions` | Дополнительные условия. | `array` | Нет | Массив строк. |
| `required_documents` | Обязательные документы. | `array` | Нет | Массив строк. |
| `choosable_documents` | Документы на выбор. | `array` | Нет | Массив строк. Пример: `"choosable_documents": ["Справка 2-НДФЛ, Справка от работодателя","Подтверждение на Госуслугах"]`. |
| `propousal_link` | Ссылка на предложение. | `object` | Да | В объекте должна быть хотя бы одна ссылка на анкету. |
| `propousal_link.link` | Ссылка на web-сайт. | `string` | Нет |  |
| `propousal_link.deep` | Ссылка на приложение. | `string` | Нет |  |
| `props` | Дополнительные служебные параметры. | `array` | Нет | Массив строк, который может содержать любую служебную информацию, оговоренную вне данной страницы. |

В случае асинхронной модели обработки данных:

Внимание

Специалисты отдела клиентского сервиса могут вас проконсультировать только по кампаниям того логина, с которого вы обращаетесь. Логин можно увидеть, если открыть [ya.ru](http://ya.ru/) на соседней вкладке браузера. Специалист получит доступ к вашим данным только при обработке обращения.

Сканируйте QR-код или нажмите на него для перехода по ссылке.


При выборе Telegram, WhatsApp учитывайте, что Яндекс не контролирует, как сторонние мессенджеры хранят ваши данные и переписку на своей стороне, и не несет за это ответственность.
