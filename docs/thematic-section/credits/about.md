# Кредиты — API для партнеров

- [Как настроить](../../thematic-section/credits/about.md#setup)
- [Обзор ресурсов](../../thematic-section/credits/about.md#list)
- [Схема взаимодействия через API](../../thematic-section/credits/about.md#scheme)

Используйте возможности API для передачи Яндексу информации:

- о предложениях по кредитам и кредитным картам, их стоимости, ежемесячному платежу;
- об оформлении заявок на получение займа, совершенном после перехода с универсальных ответов в результатах поиска Яндекса на ваш сайт.

Для корректной интеграции необходимо реализовать публичный API эндпоинт.

## Как настроить

Для подключения в качестве партнера:

1. Свяжитесь со службой поддержки по адресу [finance-search@yandex-team.ru](mailto:finance-search@yandex-team.ru). В заголовке письма напишите: «Интеграция API в кредитах <Название организации>».
2. Дождитесь, когда менеджер свяжется с вами по оставленным в форме контактам.
3. Чтобы предоставить необходимую информацию, реализуйте публичный API эндпоинт. К нему Яндекс будет отправлять запросы. Вам нужно будет предоставлять офферы в ответ.
4. Пройдите проверку работоспособности интеграции.

## Обзор ресурсов

Конечная интеграция с банком-партнером осуществляется посредством реализации HTTP/REST-метода.

Для реализации API используйте:

```
https://host/integrations
```

После реализации API нужно настроить доступы, чтобы эндпоинты были доступны без VPN из внешнего контура. Яндекс готов предоставить диапазон IP-адресов для выполнения запросов к партнерскому API.

Ниже приведен список ресурсов API и возможные операции над ними:

|  |  |  |  |
| --- | --- | --- | --- |
| **URI** | **GET** | **POST** | **DELETE** |
| Работа с предложениями банков |  |  |  |
| `api/banking/loan_approval` |  | [Получить все предложения банков](../../thematic-section/credits/post-banking.md) |  |

## Схема взаимодействия через API

После подтверждения действия пользователя, например на выдачу кредита, направляется HTTP/REST запрос на сервис интеграции, реализованный на стороне партнера для получения информации о предложениях или отказе.
