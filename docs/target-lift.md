# Target Lift

Target Lift — это исследование, которое позволяет оценить как росло число целевых действий на сайте после просмотра медийной рекламы.

Исследование Target Lift доступно для новых медийных кампаний с [баннерами](./products-cpm-campaign/about.md), [видео](./products-cpm-campaign-video/about.md), в том числе непропускаемыми, [фиксированным CPM](./fixed-cpm-campaigns/products-fixed-cpm.md) и [видеобаннерами](./products-cpm-campaign-videobanner/about.md).

Исследование Target Lift недоступно, если в кампании подключена стратегия «Ручное управление ставками».

В параметрах кампании в блоке **Целевые действия** должны быть указаны цели, которые будут использоваться для работы стратегии и в дальнейшем для аналитики.

В качестве целей указывайте те события, достижение которых приносит пользу вашему бизнесу. Рекомендуется использовать те же цели, которые вы используете для performance-рекламы при подключении конверсионных стратегий.

Примечание

Исследования рекламы на устройствах Connected TV находятся на этапе бета-тестирования. Результаты исследований зависят от объема полученных данных и связаны с особенностями кросс-девайс склейки.
