# Гид по интерфейсу

Для представителей агентств в Директе доступны специальные страницы и дополнительные возможности:

### Мои клиенты

На странице перечислены данные о ваших клиентах: логин, ФИО или название паспортной организации, контакты и список рекламных кампаний. Как добавить информацию о [новом клиенте](ru/agencies/manage-clients#new-client).

Главный представитель и представители видят данные о всех клиентах агентства, представители по работе с клиентами — только данные своих клиентов.

### Администрирование

На странице находятся основные настройки:

- **Параметры представителя агентства** — различные настройки представителя, например, уведомления, контакты представителя, которые можно показывать клиенту.
- **Все кампании моих клиентов** — список всех кампаний клиентов представителя.
- **Статистика по клиентам агентства** — сводка суммарных показателей кампаний по каждому из клиентов.
- **Скопировать кампанию** — [копирование кампаний](https://direct.yandex.ru/dna/copy-campaigns) в рамках кампаний одного клиента.
- **Перенос средств** — инструмент, который позволяет перемещать средства между клиентами агентства.
- **Представители агентства** — информация обо всех представителях агентства, доступная только главным представителям. [Подробнее о системе представителей](ru/agencies/representatives).
- **Назначение клиентов представителям агентства** — раздел для назначения представителям агентства клиентов, с которыми они будут работать. [Подробнее о назначении клиентов представителям](ru/agencies/representatives#assign-clients).

Внимание

Специалисты отдела клиентского сервиса могут вас проконсультировать только по кампаниям того логина, с которого вы обращаетесь. Логин можно увидеть, если открыть [ya.ru](http://ya.ru/) на соседней вкладке браузера. Специалист получит доступ к вашим данным только при обработке обращения.

Сканируйте QR-код или нажмите на него для перехода по ссылке.


При выборе Telegram, WhatsApp учитывайте, что Яндекс не контролирует, как сторонние мессенджеры хранят ваши данные и переписку на своей стороне, и не несет за это ответственность.
