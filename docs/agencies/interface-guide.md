# Гид по интерфейсу

Для представителей агентств в Директе доступны специальные страницы и дополнительные возможности:

### Мои клиенты

На странице перечислены данные о ваших клиентах: логин, ФИО или название паспортной организации, контакты и список рекламных кампаний. Как добавить информацию о [новом клиенте](../agencies/manage-clients.md#new-client).

Главный представитель и представители видят данные о всех клиентах агентства, представители по работе с клиентами — только данные своих клиентов.

### Администрирование

На странице находятся основные настройки:

- **Параметры представителя агентства** — различные настройки представителя, например, уведомления, контакты представителя, которые можно показывать клиенту.
- **Все кампании моих клиентов** — список всех кампаний клиентов представителя.
- **Статистика по клиентам агентства** — сводка суммарных показателей кампаний по каждому из клиентов.
- **Скопировать кампанию** — [копирование кампаний](https://direct.yandex.ru/dna/copy-campaigns) в рамках кампаний одного клиента.
- **Перенос средств** — инструмент, который позволяет перемещать средства между клиентами агентства.
- **Представители агентства** — информация обо всех представителях агентства, доступная только главным представителям. [Подробнее о системе представителей](../agencies/representatives.md).
- **Назначение клиентов представителям агентства** — раздел для назначения представителям агентства клиентов, с которыми они будут работать. [Подробнее о назначении клиентов представителям](../agencies/representatives.md#assign-clients).
