# Как оплатить размещение клиента

Если агентство работает по [оферте](https://yandex.ru/legal/oferta_direct/), то ему доступны все стандартные [способы оплаты](ru/payments/payment-methods).

Для агентств, [работающих по договору с Яндексом](https://yandex.ru/adv/partners/cooperation-contract), предусмотрен только один вид оплаты — банковский перевод.

Агентство самостоятельно выставляет счет на аккаунт клиента, на который нужно зачислить средства, и оплачивает его со своего расчетного счета.

Чтобы сформировать счет:

1. На странице выбранного клиента в меню слева нажмите **Пополнить**. Чтобы кнопка **Пополнить** стала доступна, у клиента уже должна быть создана хотя бы одна кампания.
2. Укажите сумму, которую нужно внести на счет. После этого сформируется счет для оплаты.
3. Нажмите **Выписать счет**.

Сформированный счет можно распечатать, загрузить в формате DOC/DOCX или отправить по электронной почте. Распечатать счет возможно только на этой странице.

Деньги поступят на общий счет аккаунта клиента в течение 1-2 дней после оплаты.

Обратите внимание, что в платежном поручении в поле **Назначение платежа** нужно указать номер счета, иначе идентифицировать платеж будет невозможно.

Если у вас возникли другие вопросы по оплате, обратитесь к нам через [форму обратной связи](https://yandex.ru/partner-office/knowledge-base/feedback-forms/finance) по финансовым вопросам для агентств.

Сканируйте QR-код или нажмите на него для перехода по ссылке.
