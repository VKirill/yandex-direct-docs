# Повышение эффективности рекламной кампании

Чтобы добиться лучших результатов рекламной кампании, вы можете:

- уточнить [ключевые фразы](../keywords/building-keyword-list.md) и добавить новые;
- проработать [элементы объявления](../efficiency/improve-your-ads.md);
- изменить [ставки по ключевым фразам](../strategies/manual-strategy.md#how-to-bid);
- указать [регионы](../efficiency/geotargeting.md), в которых показывается объявление;
- настроить [корректировку ставок по времени](../efficiency/timetargeting.md#hourly-adjustment);
- создать [отдельную кампанию для показов в сетях](../efficiency/content-websites.md);
- провести [эксперименты с объявлениями](../efficiency/ad-groups.md).

Проанализируйте [статистику](../statistics/performance-stat-guide.md) по своей рекламной кампании, чтобы увидеть ее сильные и слабые стороны и выбрать наилучший способ увеличения эффективности.

Обратите внимание на качество вашего сайта. Выяснить причины ухода посетителей и понять, как улучшить результат, можно при помощи [Яндекс Метрики](../statistics/metrika.md).
