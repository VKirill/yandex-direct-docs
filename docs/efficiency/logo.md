# Логотип

Логотип можно добавлять в [видеообъявления](../products-cpm-campaign-video/about.md) медийных кампаний. Он повысит заметность бренда ещё до начала просмотра ролика.

**Пример готового объявления с дополнительными элементами**

## Как добавить

Логотип можно добавить при создании или редактировании объявления в блоке **Логотип**.

Для логотипа подходит квадратное изображение от 80 пикселей по стороне, размером до 10 МБ, в формате JPG, PNG или GIF (будет использован только первый кадр).

Загрузите картинку с компьютера или выберите из ранее добавленных в Директ. Изображение на логотипе должно соответствовать [требованиям](../moderation/ad-rules.md#pics). Проверьте, как логотип отображается в объявлении. Если он не считывается, лучше заменить картинку.

Статистика показов объявлений с дополнительным оформлением доступна в [Мастере отчетов](../statistics/report-wizard.md). Чтобы проанализировать количество кликов по логотипу, тексту, выберите срез **Место клика** → **Заголовок**.
