Вы можете настроить сбор данных о работе пользователей с Турбо-сайтами в Яндекс Метрике с помощью [автоматически созданного](ru/efficiency/turbo-pages-statistics#auto) или вашего [собственного](ru/efficiency/turbo-pages-statistics#user) счетчика, а также через [добавление специальных параметров к ссылке](ru/efficiency/turbo-pages-statistics#tags). Посмотреть статистику по Турбо-сайтам можно в [Мастере отчетов](ru/efficiency/turbo-pages-statistics#report-wizard).

## Яндекс Метрика

Для Турбо-сайта вы можете использовать автоматический или собственный счетчик. В любой момент вы можете заменить автоматический счетчик пользовательским и наоборот. По умолчанию на все страницы Турбо-сайта устанавливается один счетчик, но вы можете установить счетчики на отдельные страницы в настройках каждой из них.

Внимание

К счетчикам Турбо-сайта нельзя подключить опцию [Вебвизор](https://yandex.ru/support/metrica/general/counter-webvisor.html).

### Автоматический счетчик

При создании первого Турбо-сайта в вашем аккаунте [Яндекс Метрики](https://metrika.yandex.ru) автоматически появится счетчик с названием Turbo Pages. Счетчик по умолчанию будет использоваться во всех Турбо-сайтах вашего аккаунта. С его помощью можно проанализировать эффективность Турбо-сайтов и понять, как их улучшить. Счетчик доступен только для просмотра.

К счетчику будут привязаны цели Яндекс Метрики. Полный список целей находится ниже.

Список целей автоматического счетчика

| Шапка — нажатие пункта меню | В секции **Шапка** нажат любой пункт меню |
| Шапка — нажатие кнопки | В секции **Шапка** нажата кнопка перехода на сайт или к другой секции Турбо-страницы |
| Шапка — нажатие кнопки социальной сети (Вконтакте / Viber / Одноклассники / WhatsApp / Tumblr / Pinterest / YouTube / Telegram) | В секции **Шапка** нажата кнопка социальной сети (Вконтакте / Viber / Одноклассники / WhatsApp / Tumblr / Pinterest / YouTube / Telegram) |
| Отправка заявки из формы | Нажата кнопка отправки заявки из формы |
| Загрузка страницы благодарности | Загружена страница благодарности |
| Переход на сайт со страницы благодарности | Нажата кнопка перехода на сайт на странице благодарности |
| Возврат на турбо-страницу со страницы благодарности | Нажата кнопка возврата на странице благодарности |

### Пользовательский счетчик

Вы можете использовать собственный счетчик Яндекс Метрики. Это позволит самостоятельно настроить цели и привязать их только к тем элементам Турбо-сайта, для которых вы хотите отслеживать действия пользователей. Мы рекомендуем использовать счетчик основного сайта, но вы можете создать отдельный счетчик для Турбо-сайта. Можно использовать счетчик, к которому у вас настроен [гостевой доступ на редактирование](http://yandex.ru/support/metrica/general/access.html#guest).

Чтобы указать собственный счетчик, нажмите  → **Аналитика**. В выпадающем списке выберите **Пользовательский** и укажите нужный счетчик.

К кликабельным элементам Турбо-сайта вы можете привязать одну или несколько целей пользовательского счетчика. В поле **Цели Яндекс Метрики** введите их идентификаторы через запятую. Указанные идентификаторы должны совпадать с идентификаторами целей в Яндекс Метрике. Для формы заявки и кнопки обратной связи типа «Телефон» обязательно нужно указать хотя бы одну цель. Подробнее о целях смотрите [в Справке Яндекс Метрики](https://yandex.ru/support/metrica/general/goals.html).

## Мастер отчетов

Оценить результат работы объявлений с Турбо-сайтами можно в [Мастере отчетов](ru/statistics/report-wizard). Например, чтобы посмотреть, как уменьшилось количество отказов при показе на мобильных, используйте срез **№ объявления**. В блоке **Столбцы** выберите **Отказы**. Затем добавьте фильтр **Тип устройства** → **мобильные**.

[Подробнее о том, как оценить результат кампании](ru/statistics/performance-stat-guide)

## Добавление параметров к ссылке

К ссылке на Турбо-сайт можно добавить различные параметры и метки, чтобы системы веб-аналитики (например, Яндекс Метрика) получали дополнительную информацию о переходах по этой ссылке. На странице редактирования объявлений нажмите **+Трекинговые параметры**. Параметры будут добавлены ко всем ссылкам на Турбо-сайты, в том числе и к быстрым ссылкам. Подробнее о [параметрах URL](ru/statistics/url-tags).
