# Доставка

- [Как настроить](../efficiency/delivery.md#kak-nastroit)
- [Как отображается в объявлениях](../efficiency/delivery.md#kak-otobrazhaetsya-v-obuyavleniyah)

Информация о доставке может быть добавлена к объявлению в Рекламной сети Яндекса: товарному, текстово-графическому или объявлению для страниц каталога. Элемент содержит краткое описание условий доставки и выделяется особым значком. Он помогает подчеркнуть преимущества вашего предложения и делает объявления привлекательнее. Добавить информацию о доставке можно в Единой перфоманс-кампании. Для одной кампании можно указать информацию только об одной доставке.

## Как настроить

В настройках Единой перфоманс-кампании в блоке **Дополнительные элементы объявлений** выберите **Доставка** → **Добавить**. Укажите хотя бы один из параметров: **Тип доставки**, **Стоимость** или **Срок**.

- **Тип доставки** — выберите подходящий вариант:

  - «Курьерская»;
  - «Самовывоз» — из ПВЗ или из магазина;
  - «Без уточнений» — например, доступны разные типы доставки с одинаковыми условиями (сроком, стоимостью). Также этот вариант можно использовать, если вы хотите сделать акцент на других характеристиках доставки.
- **Стоимость** — выберите подходящий вариант:

  - «Без уточнения» — вы можете не уточнять стоимость доставки, если она варьируется в зависимости от заказа;
  - «Бесплатно»;
  - «Бесплатно от суммы покупки» — выберите валюту и укажите минимальную сумму покупки;
  - «Стоимость от» — выберите валюту и укажите минимальную стоимость доставки любых заказов;
  - «Фиксированная стоимость» — выберите валюту и укажите стоимость доставки любых заказов.
- **Срок** — выберите один из доступных вариантов, укажите свой период или не заполняйте это поле.

  Ограничение на ручное указание срока доставки: от одного до 365 дней. Как будет выглядеть итоговый текст, можно посмотреть в превью в интерфейсе на странице объявлений.
- **Регионы доставки** — выберите, в каком регионе осуществляется доставка.

  Если условия доставки различаются в зависимости от города, опишите их так, чтобы они были актуальны для всех регионов, или добавьте доставку только для регионов с одинаковыми условиями. Директ соотнесет местоположение пользователя, геотаргетинг, настроенный в кампании, и регион, указанный для доставки. Если они совпадают, в объявлениях кампании может быть показана информация о доставке. Если нет, информация о доставке не будет добавляться в объявления.

  > Например, если кампания настроена для Москвы и Санкт-Петербурга, а информация о доставке указана только для Москвы, пользователи в Москве увидят баннеры с деталями доставки, а пользователи из Санкт-Петербурга нет.

Настроенную доставку можно использовать в нескольких рекламных кампаниях. Для этого выберите нужную доставку из выпадающего списка. Если информацию надо изменить, ее можно отредактировать в этом же поле. Она обновится во всех кампаниях, в которых добавлена эта доставка.

## Как отображается в объявлениях

Информация о доставке может отображаться в однотоварном и многотоварных форматах объявлений в РСЯ.

**Отображение доставки в однотоварном формате объявлений**

В **многотоварных объявлениях** доставка может отображаться в формате смарт-мозаики, Shoppable Ads и Smart Fullscreen. [Подробнее об этих форматах](https://yandex.ru/adv/edu/materials/kak-prodvigat-bolshoj-assortiment-tovarov-v-rsya)

Для каждого показа в РСЯ технология [Smart Design](../efficiency/improve-your-ads.md#smart-design) автоматически создает наиболее привлекательное объявление с учетом возможностей площадки, предпочтений пользователя и прогноза конверсий. Чтобы реклама приносила максимум конверсий, добавляйте как можно больше деталей о предложении на этапе настройки, в том числе информацию о доставке.
