# Кнопка действия

Кнопка действия добавляется в [видеообъявления](../products-cpm-campaign-video/about.md) медийных кампаний или текстово-графические объявления в Единой перфоманс-кампании с изображением или видео. Она привлекает внимание и побуждает пользователей взаимодействовать с вашим объявлением. Кнопка дополняет объявление еще одной ссылкой. По клику на текст или видео пользователь переходит на одну страницу сайта, а по кнопке действия — на другую. Ссылки также могут совпадать.

## Как настроить

По умолчанию кнопка действия настраивается автоматически — нейтральный текст, например, «Узнать больше», «Перейти на сайт», и ссылка, совпадающая с ссылкой объявления.

Вы можете выбрать более продающий текст для кнопки и указать другую ссылку, когда создаете или редактируете объявление. Для этого в блоке **Кнопка** вместо текста по умолчанию выберите подходящий и укажите ссылку на страницу перехода. Она может отличаться от ссылки в объявлении. Чтобы добавить ссылку на мессенджер, используйте форматы:

- для Telegram — `t.me/username`;
- для WhatsApp — `wa.me/phone_number`.

Добавить кнопку в несколько текстово-графических объявлений в Единой перфоманс кампании можно с помощью [массовых действий](../campaigns/multi-edit.md).

Статистика показов объявлений с дополнительным оформлением доступна в [Мастере отчетов](../statistics/report-wizard.md). Чтобы проанализировать количество кликов по кнопке, выберите срез **Место клика** → **Кнопка**.
