# Как добавить Турбо-сайт

При создании или редактировании объявлений Турбо-сайт можно добавить к ссылке на основной сайт, [быстрой ссылке](../efficiency/quick-links.md), [виртуальной визитке](../efficiency/vcards.md) или использовать только Турбо-сайт. От этого зависит, куда будет вести реклама на разных устройствах.

Выберите, на каких устройствах вы хотите использовать Турбо-сайт:

- Только на мобильных устройствах: в блоке **Страница перехода** в разделе **На десктопах и планшетах** введите ссылку на основной сайт. В разделе **На мобильных устройствах** выберите **Турбо-страница**. В этом случае при показе на компьютерах и планшетах ваша реклама будет вести на основной сайт, на смартфонах — на Турбо-сайт.
- На всех устройствах: в блоке **Страница перехода** в обоих разделах выберите **Турбо-страница**.

Нажмите кнопку **Добавить** и выберите один из вариантов:

- **Создать за минуту** — для быстрого создания Турбо-страницы. Достаточно заполнить несколько полей формы, и страницу можно опубликовать и сразу привязать к объявлению.
- **Создать из шаблона** — для создания страницы с готовой структурой, которая учитывает особенности вашего бизнеса.
- **Мои Турбо-страницы** — для перехода в библиотеку Турбо-сайтов. Нажмите **Создать** → **Турбо-сайт** или выберите ранее добавленный сайт.

Подробнее [о том, как создать Турбо-сайт в конструкторе](../efficiency/turbo-pages-builder.md).

## Массовое управление

Добавить, заменить или удалить Турбо-сайты сразу в нескольких объявлениях можно с помощью массовых изменений. На странице объявлений выделите нужные объявления и в списке действий выберите **Турбо-страницы**. На открывшейся странице выберите действие и нажмите кнопку **Применить**.
