# Как запустить кампанию

Нажмите **Добавить** → **Кампанию**. На вкладке **Режим эксперта** нажмите **Охватные кампании**.

- Чтобы создать кампанию с видеороликом, выберите **Видео**.
- Чтобы создать кампанию с видеороликом, который нельзя пропустить, выберите **Непропускаемые видео**. Такое размещение работает для видеороликов до 15 с и приносит максимальный досмотр, но делает видеорекламу дороже. Непропускаемое видео размещается только в InStream.

#### Шаг 1. Настройте параметры кампании

1. Укажите название кампании и ссылку на сайт. Директ автоматически найдет счетчики Метрики, установленные на сайте.
2. Выберите стратегию показа объявлений. Мы рекомендуем автоматические стратегии:

   - Чтобы получить максимум просмотров за выбранную стоимость — [Оплата за просмотры](../strategies/maximum-video-views.md).
   - Для вывода на рынок новых товаров и услуг — [Снижение цены повторных показов](../strategies/decrease-price-for-repeat-impressions.md).
   - Для продвижения товаров и услуг среди целевой аудитории — [Максимум показов по минимальной цене](../strategies/maximize-impressions-for-lowest-price.md).
   - Рост доли поисков бренда и его продуктов после показа медийной рекламы — [Рост запросов в Поиске](../strategies/optimize-queries-in-search.md). Создать кампанию с этой стратегией можно только при запуске исследования [Search Lift](../search-lift.md).
   - Увеличение количества визитов на сайт после показа медийной рекламы — [Рост визитов на сайте](../strategies/growth-of-site-visits.md). Создать кампанию с этой стратегией можно только при запуске исследования [Visit Lift](../visit-lift.md).
   - Увеличение количества целевых действий на сайте после показа медийной рекламы — [Рост целевых действий на сайте](../strategies/growth-of-conversions.md). Создать кампанию с этой стратегией можно только при запуске исследования [Target Lift](../target-lift.md).
   - Увеличение прироста брендовых метрик после показа медийной рекламы — [Рост брендовых метрикβ](../strategies/growth-of-brand-metrics.md). Создать кампанию с этой стратегией можно только при наличии нового прикрепленного исследования [Brand Lift](../brand-lift.md)

   Выберите цель показов: досмотры или переходы на сайт. После создания кампании изменить настройки оптимизации показов нельзя.

   Укажите срок распределения бюджета — период или неделя, бюджет и среднюю цену за тысячу показов либо цену за просмотр, если выбрана стратегия «Оплата за просмотры».
3. В блок **Целевые действия** автоматически будут добавлены цели со счетчиков Метрики. При необходимости измените список целей, которые выбрал сервис: удалите или добавьте новые. Если на счетчиках нет целей, сервис предложит их добавить в Метрике.
4. Задайте сроки проведения кампании, [расписание показов](../efficiency/timetargeting.md) и [ограничение частоты показов](../campaigns/campaign-settings.md#frequency-capping).
5. Если кампания подходит под условия [Brand Lift](../brand-lift.md), [Search Lift](../search-lift.md), [Visit Lift](../visit-lift.md) или [Target Lift](../visit-lift.md), запустите исследования.

#### Шаг 2. Настройте целевую аудиторию

1. Выберите [регионы показа](../efficiency/geotargeting.md).

   Если регион показа не Россия, выберите язык объявлений — так объявления будут показываться только тем потенциальным клиентам, которые понимают выбранный язык или посещают интернет-ресурсы на этом языке.
2. Настройте сбор аудитории для ретаргетинга. Для каждого выбранного события автоматически будет создан отдельный сегмент в [Яндекс Аудиториях](../technologies-and-services/audience-segments-in-direct.md) на том же аккаунте, что и кампания в Директе. В него попадут пользователи, которые начали смотреть или посмотрели 25%, 50% или 75% хотя бы одного видео из группы (в зависимости от выбранного события).

   Подробнее в разделе [Ретаргетинг на аудиторию видео](../products-cpm-campaign-video/retargeting.md).
3. Выберите условие показа. После сохранения группы изменить тип условий показа нельзя.

   - [Профиль пользователей](../products-cpm-campaign/display-ads-settings.md) — совокупность характеристик пользователей, которым вы хотите показывать рекламу. Например, пол, возраст, интересы в интернете. Определите портрет именно вашей аудитории и, исходя из этих данных, сформируйте профиль.
   - [Жанры и тематики](../products-cpm-campaign/genres-and-themes.md) — показы в потоковых видео указанных жанров и тем. Например, реклама ресторана может быть показана внутри ролика о кулинарии.
4. Добавьте корректировки.
5. Оцените, сколько пользователей сможет увидеть объявления из группы, с помощью [прогноза охвата](../products-cpm-campaign/mediaplan.md).

#### Шаг 3. Настройте объявление

**Укажите ссылку на страницу сайта или соцсети**

Дополнительная информация о предложении будет открываться по клику на баннер. Подробнее об [отображаемых ссылках](../efficiency/display-link.md).

**Создайте видеоролик**

Для создания объявления используйте собственный видеоролик. Необязательно нанимать профессионалов или арендовать студию. Вы можете снять ролик на смартфон с хорошей камерой.

Видео должно соответствовать [правилам модерации](../moderation/ad-rules.md#video) и следующим техническим требованиям:

|  |  |
| --- | --- |
| Формат | MP4, WebM, MOV, QT, FLV, AVI |
| Размер | Не более 100 МБ |
| Длительность | От 5 до 60 c  Для непропускаемого видео можно использовать видеоролики только до 15 с. |
| Соотношение сторон | 16 : 9; 1 : 1; 9 : 16 |
| Минимальное разрешение | 360 p |
| Рекомендованное разрешение | 1080 p |
| Частота кадров | от 20 до 30 кадров в секунду |
| Видеокодеки | H.264, VP6F, VP8, Theora |
| Количество видеодорожек | 1 |
| Минимальное качество | 44 кГц, 16 Кбит/c, стерео |
| Аудиокодеки | AAC, MP3, Vorbis |
| Количество аудиодорожек | 0 или 1 |
| Кодек файла | avc1, F4V, isom, M4A, M4V, mp4, mp41, mp42, qt |
| Обложка | Нет |

Учитывайте дизайн рекламного плеера при подготовке видео — расположение и формат кнопки перехода на сайт. Она закреплена в нижнем левом углу ролика, и ее цвет и положение нельзя изменить:

- Лучше размещать основной рекламный контент в центральной части видео, отступая 20% снизу.
- При необходимости размещайте предупреждение внизу видео, в нижних 10%, чтобы кнопки его не перекрывали.

Юридическую информацию не обязательно вставлять в ролик на этапе монтажа. Укажите предупреждение при загрузке видео в блоке **Юридическая информация**, и оно будет доступно в меню в правом верхнем углу видео.

**Как создать видео в конструкторе**

1. В блоке **Видео** нажмите **Конструктор**. Откроется окно конструктора видео.

2. Загрузите видео с компьютера, по ссылке или выберите из созданных в видеоконструкторе. Аудио добавится из загруженного видеоролика автоматически.

Обработка видеоролика займет несколько минут. Если ролик не соответствует техническим требованиям, он не загрузится. Вы увидите [сообщение об ошибке](../products-cpm-campaign-video/video-clip-errors.md). Исправьте ее и загрузите ролик заново.

3. Добавьте пэкшот — изображение, которое будет появляться на несколько секунд после вашего видео. Он помогает закрепить образ рекламируемого объекта у зрителя. Для пэкшота можно использовать, например, фото товара или логотип компании. Кликнув на пэкшот, пользователь перейдет на ваш сайт.

Требования к пэкшоту

|  |  |
| --- | --- |
| Формат | JPEG |
| Ширина изображения | от 1080 до 3840 пикселей |
| Высота изображения | от 1080 до 3840 пикселей |
| Соотношение сторон | Совпадает с соотношением сторон видео |

4. Выберите текст для кнопки перехода.

5. Если требуется, установите возрастную метку. В блоке **Юридическая информация** укажите данные, необходимые по [требованиям законодательства и правилам Яндекса](../moderation/adv-rules.md). В объявлении они будут доступны в меню в правом верхнем углу видео.

6. Нажмите кнопку **Создать**. Видео будет добавлено в объявление.

Примеры размещения сейфзон в зависимости от формата видео

- Вертикальный
- Горизонтальный
- Квадратный

В основном сейфзоны расположены, как на примерах выше. Но бывают исключения.

**Добавьте к видеоролику субтитры**

Ваше предложение может заинтересовать тех, кто смотрит видео без звука. Субтитры будут отображаться на всех [форматах](../products-cpm-campaign-video/about.md) видео, они не будут перекрываться кнопками или другими элементами оформления. Выберите способ создания.

- **Автоматические**. Сгенерируйте субтитры — в среднем это занимает меньше минуты. Исправьте их, если необходимо: скачайте файл, измените его и загрузите обратно.
- **Из файла**. Загрузите файл с субтитрами в формате SRT или VTT, размером не более 2 Мб.

После создания субтитры отобразятся на видео в блоке предпросмотра, чтобы была возможность оценить синхронность текста и видео.

Субтитры будут добавлены в ролик, если их отображение не перекрывает основную часть контента видео.

Видеоролики проходят модерацию, поэтому субтитры должны совпадать со словами, которые произносятся в видео.

**Продумайте оформление**

- [Логотип](../efficiency/logo.md) — повысит заметность бренда до начала просмотра ролика.
- [Кнопка действия](../efficiency/button.md) — призыв к покупке или предложение узнать больше об услугах. Пользователь будет заранее знать, куда попадет после клика, например, на страницу товара или в форму заказа. Это увеличит количество целевых переходов.
- [Заголовок и текст](../efficiency/text-and-title.md) — используйте слоган компании или информацию о специальных предложениях, например, о скидках или доставке.
- [Карусель с изображениями](../products-cpm-campaign-video/carousel-in-video-ads.md) — сократит путь клиента от рекламы до возможной покупки. Загрузите от 1 до 7 слайдов.

Текст и кнопка будут кликабельными. В тексте будет такая же ссылка, как и на видео, а для кнопки и в карусели можно задать дополнительную — на страницу товара или услуги.

Логотип, кнопка и заголовок с текстом могут появиться рядом с видео в текстовом контенте, в приложениях и видео за вознаграждение. Расположение элементов и возможность их отображения зависят от площадки и места показа. Чем больше дополнений вы добавите, тем больше вариантов объявления сможет увидеть ваша аудитория на разных площадках.

**Укажите пиксель Яндекс Аудиторий и счетчик показов**

Вы можете выстраивать различные сценарии коммуникации с аудиторией вашей медийной рекламы, используя пиксель Яндекс Аудиторий. Это специальный код, который встраивается в рекламные материалы.

Создайте пиксель в Яндекс Аудиториях, подробнее в [Справке Яндекс Аудиторий](https://yandex.ru/support2/audience/ru/pixel#pixel).

Добавьте код пикселя к баннеру. После старта показов пиксель начнет собирать *анонимные идентификаторы* пользователей, видевших вашу рекламу. Эти данные можно будет использовать при настройке ретаргетинга.

Уникальный идентификатор мобильного приложения или браузера пользователя. Идентификатор становится активным, когда пользователь работает в приложении или браузере. При обновлении сегментов неактивные анонимные идентификаторы удаляются из них и добавляются новые (к примеру, если пользователь приобрел новое устройство или переустановил приложение) — так, чтобы в сегментах оставались только анонимные идентификаторы, которые были активны в последние 30 дней.

Вы можете использовать счетчик ADFOX или пиксель [Яндекс Метрики для медийной рекламы](https://yandex.ru/support/media-metrica/) для подсчета показов объявлений. В счетчике или пикселе обязательно должен быть макрос `%random%`. Также они должны поддерживать протокол `https`.

Примечание

Если вы выбрали автоматическую стратегию и нацеливание по профилю пользователя, после сохранения групп объявлений вернитесь к настройкам стратегии. Система учтет размеры креативов и условия показа и предложит уточненную рекомендацию для средней цены, которая позволит эффективно использовать указанный бюджет за период.

Нажмите кнопку **Запустить кампанию**.

Яндекс как оператор рекламных данных при создании кампании присваивает каждому креативу уникальный идентификатор — токен. Затем по этому токену агрегирует всю информацию по размещению и передает в Единый реестр интернет-рекламы (ЕРИР). Подробнее о [маркировке рекламы](https://yandex.ru/adv/ad-labeling).

### Модерация объявлений и запуск рекламной кампании

Показы объявлений начнутся после того, как вы [оплатите размещение рекламы](../payments/payment-methods.md). Если на счете достаточно средств, кампания запустится автоматически.

Модераторы проверят, соответствуют ли объявления [требованиям](../moderation/adv-rules.md). Обычно модераторы проверяют объявления в течение нескольких часов. В выходные и праздники модерация может занять больше времени, чем в будни.

Письмо с результатами проверки придет на адрес, который вы указали при создании рекламной кампании.
