# Продвижение специалистов

- [Создание кампании](../campaign-master/service.md#launch-campaign)
- [Модерация, оплата и запуск](../campaign-master/service.md#moderation)
- [Статистика кампании](../campaign-master/service.md#statistics)

Если вы частный мастер и размещаете объявления на платформах-агрегаторах, вы можете продвигать свои услуги через кабинет Директа.

Для продвижения в [галерее услуг](../service-gallery/about.md) настройте кампанию через Мастер кампаний. Объявления будут сгенерированы на основе данных из вашего профиля: фотографий, контактов, рейтинга, отзывов и т.д.

Примечание

Если сервис, на котором вы размещаете объявления, не предоставил Директу необходимые данные о вашем профиле, создать кампанию не получится.

## Создание кампании

1. Нажмите **Добавить** → **Кампанию**. На вкладке **Мастер кампаний** выберите **Конверсии и трафик** или **Продвижение специалистов**.
2. Введите ссылку на свой профиль. Если сервис, где вы размещаете объявления, предоставил данные о вас и они прошли модерацию, Директ добавит ваш профиль в кампанию. Если профиль не подтянулся, вы можете попробовать другой профиль на сервисе из списка доступных.
3. Если в вашем профиле на сервисе указаны контакты для связи с клиентами (телефон, мессенджеры), они автоматически добавятся в кампанию.
4. Выберите [регион показа](../efficiency/geotargeting.md). Объявления будут показаны по расширенному геотаргетингу на основе поисковых запросов и по регулярному региону пользователя.

   Отключить расширенный геотаргетинг в Мастере кампаний невозможно. Но вы можете уточнить тематические запросы с учетом его работы. Подробнее см. в разделе [Как работает геотаргетинг](../efficiency/geotargeting.md#how-it-works).
5. Выберите время показа объявлений. Подробнее о [временном таргетинге](../efficiency/timetargeting.md).
6. Настройте аудиторию. Выберите один из предложенных вариантов:

   - **Подобрать оптимальную** — Директ подберет подходящие настройки на основе вашего профиля.
   - **Настроить вручную**:

     - Задайте *таргетинги* по [профилю пользователя](../products-cpm-campaign/display-ads-settings.md): пол, возраст и тип устройств, на которых будут показаны объявления.
     - В блоке **Интересы и поисковые запросы** укажите [интересы и привычки](../products-text-image-ads/user-profile.md), например сайты, компании или темы, которые может искать ваша аудитория. А также [тематические слова](../keywords/keywords.md) — ключевые фразы, по которым Директ подберет заинтересованных пользователей.
     - Нажмите **Добавить исключения**. Показы по автотаргетингу включены по умолчанию и выбраны все [категории](../impression-criteria/autotargeting.md#categories). Задайте исключения — [минус-слова](../keywords/negative-keywords.md), категории автотаргетинга или упоминание брендов в запросах.

   При переходе с ручного подбора аудитории на автоматический настройки возвращаются к стандартным.
7. В блоке **Цель рекламы** выберите цель кампании: «Максимум целевых действий» или «Максимум переходов».

   При выборе цели «Максимум целевых действий» укажите счетчики Метрики для профиля. Если в вашем профиле на сервисе указаны контакты для связи с клиентами (телефон, мессенджеры), для них автоматически создастся счетчик, который будет фиксировать действия пользователя в карточке, например клик по кнопке «Позвонить», клик по кнопке «Написать в Telegram» и т.д. В блоке **Целевые действия** добавьте из списка нужные действия и укажите, сколько готовы платить за каждое из них.
8. Определите недельный бюджет:

   - Используйте один из рассчитанных системой — минимальный, оптимальный или эффективный. Прогноз строится на базе усредненных бюджетов конкурентов с учетом выбранной цели и целевых действий.
   - Задайте свой. Его должно быть достаточно для оплаты не менее десяти конверсий по самой дорогой цели.

   Директ будет расходовать средства в пределах заданной суммы. Кампания будет идти, пока на счете есть деньги.
9. Стратегия будет выбрана автоматически в зависимости от ваших настроек:

   - «Максимум целевых действий» по средней цене за неделю с одной или несколькими заданными целями — кампания принесет максимум конверсий по ним, а цена конверсии каждой из целей в среднем за неделю будет стремиться к указанной. Деньги будут списываться **за клики** по объявлению. Директ подберет аудиторию, которая с наибольшей вероятностью совершит нужные целевые действия.
   - «Максимум целевых действий» по фиксированной цене с несколькими заданными целями — кампания принесет максимум конверсий по всем указанным целям. Деньги будут списываться **за конверсии**. При достижении нескольких целевых действий, будет списана оплата за каждое из них. Директ подберет аудиторию, которая с наибольшей вероятностью совершит нужные целевые действия.
   - «Максимум целевых действий» без ограничений с одной заданной целью — кампания принесет максимум конверсий по выбранной цели за указанный недельный бюджет. Деньги спишутся **за клики** по объявлению. Директ подберет аудиторию, которая с наибольшей вероятностью совершит нужные целевые действия.
   - «Максимум переходов» по средней цене за неделю — деньги будут списываться за переходы в карточку специалиста, а стоимость перехода в среднем за неделю будет стремиться к указанной.
   - «Максимум переходов» без ограничений — будет выбрана стратегия, при которой Директ стремится привести в карточку специалиста максимум заинтересованных пользователей за указанный бюджет.
10. Нажмите **Создать кампанию**, чтобы отправить ее на модерацию. Чтобы закончить редактирование кампании позже, нажмите **Сохранить как черновик**.

## Модерация, оплата и запуск

После создания ваша кампания будет проверена на соответствие [требованиям](../moderation/adv-rules.md).

Результаты проверки вы получите в письме на адрес, указанный при регистрации в Директе.

Некоторые элементы кампании проходят модерацию отдельно (например, изображения). Если один или несколько элементов не пройдут модерацию, кампания запустится без них. Статус модерации и комментарии к нему появятся на странице кампании. Вы можете удалить элементы, которые не прошли модерацию, или отредактировать их. Отредактированные элементы отправятся на модерацию повторно и будут включены в кампанию, если успешно пройдут проверку.

Показы объявлений начнутся только после того, как вы [оплатите продвижение](../payments/payment-methods.md).

После запуска на странице кампании будет указана основная информация о ее настройках. Запущенную кампанию можно редактировать — для этого нажмите нужный параметр.

## Статистика кампании

Статистика отображается на странице кампании. Под блоком с информацией о настройках дана общая статистика кампании, ниже — более подробная статистика по отдельным параметрам.

Отображение общей статистики кампании зависит от выбранной цели:

- в кампаниях, нацеленных на переходы, будет видна статистика показов, кликов и расходов;
- в кампаниях с конверсионными стратегиями — статистика показов, кликов, конверсий, стоимости конверсии и расходов.

На графиках отражена статистика по следующим параметрам:

- пол и возраст аудитории, которая увидела ваше объявление;
- регионы показа;
- тип устройств;
- тематические фразы, по которым вас чаще всего находили.

Примечание

Возможна ситуация, когда значение параметра равно нулю. Например, доля одного из регионов показа — 0%, однако для него есть статистика показов. Это означает, что объявление было показано в регионе, но показы не привели к переходу на сайт или целевому действию на сайте.

В блоке статистики по категориям автотаргетинга указаны поисковые запросы, по которым было показано объявление.
