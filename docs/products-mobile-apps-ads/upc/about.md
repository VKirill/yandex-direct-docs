# Создание объявлений в ЕПК

Единая перфоманс-кампания (ЕПК) — инструмент для создания кампании, которая расширяет возможности привычных кампаний Директа. Позволяет осуществить более качественную настройку продвижения мобильных приложений.

Поддерживаются мобильные приложения из App Store, Google Play и RuStore.

## Преимущества

**Общая структура настроек**

Структура ЕПК выстроена по понятной и привычной логике:

- **На уровне кампании задаются общие настройки**: цель кампании (привлечение новых пользователей или ретаргетинг), места показа (только поиск, только РСЯ, или все площадки), стратегия, сроки проведения кампании, расписание, установка приложений из альтернативных магазинов (для Android), включение опции **SKAdNetwork** (для IOS) и дополнительные настройки, которые можно применить для всех групп кампании.
- **На уровне группы задаются таргетинги на целевую аудиторию**: география показов, автотаргетинг, тематические слова, условия ретаргетинга и подбор аудитории, тип контента площадок, а также настройки, которые можно применить для всех объявлений в группе.
- **На уровне объявлений есть все для работы с их внешним видом и посадочными страницами**: поля для загрузки текстов, изображений, видео, поля для настройки трекинговых ссылок, диплинков и кастомных страниц, а также доступны опции для управления атрибуцией по показам.

**Создание единой укрупненной кампании для больших охватов, эффективности и лучшего обучения стратегий**

Работа с группами в Единой-перфоманс кампании позволит тестировать наборы креативов для каждого сегмента аудитории, а также объединять объявления в укрупненные кампании, чтобы аккумулировать данные для обучения стратегий.

**Гибкое управление ценой клика, установки или in-app события**

На уровне группы доступна дополнительная корректировка, которая позволяет влиять на целевое значение цены клика, установки или события внутри приложения для конкретной аудитории.

**Статистика**

В Мастере отчетов доступна подробная статистика со всеми срезами, включая разбивку на типы объявлений.
