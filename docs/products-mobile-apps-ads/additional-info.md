Продвижение приложений по умолчанию содержит дополнительную информацию: рейтинг и количество оценок, цену, иконку, метку о возрастном ограничении. Информация загружается из магазина приложений автоматически и обновляется раз в сутки. Также вы можете выбрать текст для кнопки действия на объявлении.

## Рейтинг и количество оценок

Рейтинг в объявлении отображается в виде звездочек и округляется с точностью до 0,5. Например, если рейтинг вашего приложения 4,3, то в объявлении рейтинг будет 4,5, а если 4,2 — то 4.

Обратите внимание, что система получает данные для той страны, которая указана в параметрах ссылки. Если в параметрах ссылки нет указания страны, то данные будут загружены для страны по умолчанию — России.

## Цена

Если вы добавили цену, то она будет указана на кнопке действия, например: . В зависимости от формата блока для размещения объявлений на той или иной тематической площадке кнопка действия может быть показана без цены.

Пользователю будет показана цена в валюте страны, в которой он находится. Если при загрузке данных из магазина приложений не удалось получить цену для этой страны, то кнопка действия будет показана без цены.

## Иконка

Если иконка приложения не соответствует [требованиям модерации](../moderation/ad-rules.md#icons), объявление будет показано без иконки или с предыдущей версией иконки, которая успешно прошла модерацию. Иконка может быть пропорционально уменьшена в соответствии с настройками блока для размещения объявлений (до 16 пикселей по любой стороне).

## Метка о возрастном ограничении

Возрастное ограничение устанавливается автоматически на основе данных из магазина приложений. Если ограничения из магазина нет среди вариантов, доступных в Директе, будет установлено более строгое ограничение.

> Например, если в магазине приложений указана возрастная категория 3+, то в Директе будет установлено возрастное ограничение 6+.

Ограничение можно изменить самостоятельно.

## Кнопка действия

Текст на кнопке, отображаемой в объявлении, можно выбрать из нескольких вариантов. Если вы выбрали **Купить / Бесплатно**, значение будет выбрано автоматически: **Купить** — для платных приложений, **Бесплатно** — для бесплатных.
