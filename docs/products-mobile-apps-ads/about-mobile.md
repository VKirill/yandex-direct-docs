## Преимущества

**Доступ к качественной мобильной аудитории**

Объявления показываются на мобильных устройствах в поиске Яндекса и в самой большой партнерской сети рунета — Рекламной сети Яндекса. Вы получите новых пользователей и сможете увеличить количество установок, а также выстраивать цепочки коммуникации с вашими клиентами с помощью ретаргетинга и увеличивать количество повторных целевых действий.

**Форматы объявлений для продвижения приложений**

В одной кампании могут быть разные объявления: текстовые, объявления с изображением, графические объявления и объявления с видео. Рекомендуем использовать как можно больше вариантов, чтобы объявления показывались на всех доступных площадках.

[Рекомендации по оформлению объявлений](../products-mobile-apps-ads/recommendations.md)

**Выбор стратегии**

Специальные стратегии показа продвижения приложений работают не только на повышение количества установок, но и на выполнение пользователем целевых действий внутри приложения.

**Точный таргетинг**

Для поиска целевых пользователей вы можете использовать разные способы подбора аудитории: ключевые фразы, готовые сегменты аудиторий, автотаргетинг, аудитории ретаргетинга, а также корректировки — с их помощью можно отсечь нерелевантных пользователей.

Вы можете настраивать портрет целевой аудитории по разным признакам: пол и возраст, география, платежеспособность и другое.

## Способы запуска

- В [Единой перфоманс-кампании](../products-mobile-apps-ads/upc/about.md) — выбирайте этот тип кампании, чтобы иметь доступ к использованию расширенных возможностей режима эксперта для более гибкого управления кампаниями для продвижения приложений.

## Типы объявлений

- **Комбинаторное** — тип объявления, в котором вы указываете варианты заголовков, текстов, изображений и видео, а Директ собирает из них множество текстово-графических объявлений. Можно запустить в Мастере кампаний и в Единой перфоманс-кампании.
- **Графическое** — тип объявления, с помощью которого вы можете взаимодействовать с аудиторией через изображение. Можно запустить только в Единой перфоманс-кампании.
- **Playable Ads** — тип объявления, с помощью которого вы можете взаимодействовать с аудиторией через динамический HTML5-баннер. Можно запустить в Мастере кампаний и в Единой перфоманс-кампании.
- **Товарное объявление** — это тип объявления, которое создается из фида. Фид содержит информацию о товарах (например, название, изображение, цена) и трекинговую ссылку. Если добавить к трекинговой ссылке отложенный диплинк, пользователь после установки приложения сразу попадет на карточку товара. Можно запустить только в Единой перфоманс-кампании.
