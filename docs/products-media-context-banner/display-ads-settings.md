# Настройка условий показа

На поиске Яндекса баннеры могут быть показаны по запросам, целиком содержащим заданные вами [ключевые фразы](../keywords/keywords.md). Например, если вы задали для группы баннеров фразу *туры в космос*, то ваши баннеры могут быть показаны по запросам *туры в космос*, *горячие туры в космос*, но не будут показаны по запросам *туры* или *космос*.

При показе система учитывает разные формы слова в рамках одной части речи, в том числе формы единственного и множественного числа. Например, по фразе *путешествия марс* баннер может быть показан, если пользователь спрашивает: *путешествиЕ марс* или *путешествиЯ марс*.

Подробнее о [ключевых фразах](../keywords/keywords.md).
