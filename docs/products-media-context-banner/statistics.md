# Статистика показов

Данные статистики показов помогут вам оценить эффективность баннеров. После того как начнутся показы, вы можете посмотреть отчеты о работе рекламной кампании в **Мастере отчетов**.

> Например, вы можете посмотреть запросы, по которым были показаны ваши объявления. В блоке **Срезы** выберите **№ баннера** и **Поисковый запрос**, в блоке **Столбцы** — **Клики** и **Отказы**. Чтобы исключить показы по запросам, которые кажутся вам неподходящими, используйте [минус-фразы](../keywords/negative-keywords.md). Их можно добавить на странице редактирования кампании или группы баннеров.

Подробнее о том, [как оценить результат кампании](../statistics/performance-stat-guide.md).
