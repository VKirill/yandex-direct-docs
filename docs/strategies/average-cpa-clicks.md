# Максимум конверсий с оплатой за клики

При выборе оплаты за клики вы платите за каждый клик по объявлению.

## Настройки стратегии

1. Выберите условия и ограничение расходов, при котором вы хотите получить максимум целевых действий:

   **Задайте бюджет**

   Укажите сумму и выберите ограничение: в неделю или на период. Недельный бюджет подойдет для длительных кампаний без точной даты окончания. Если нужно контролировать расходы по флайтовым кампаниям, используйте бюджет на период.

   Минимальный бюджет:

   - На неделю — 300 рублей (*значения для других валют*).
   - На период — 50 рублей в день (*значения для других валют*).

   Подробнее о [бюджете в стратегии](ru/strategies/budgets-in-strategies).

   **Задайте ограничение расхода**

   - **Доля рекламных расходов**

     Стратегия позволяет получить максимальное количество конверсий по выбранным целям, придерживаясь при этом заданной доли рекламных расходов. Ставки подбираются таким образом, чтобы достичь наилучшего результата, а расходы на продвижение составляли бы указанный процент от дохода, который передается по выбранной цели.

     Стратегия подойдет для кампаний, которые используют электронную коммерцию или передают доход от конверсии в Метрику. Также ее можно использовать в кампаниях, где доход указан как ценность нужного целевого действия.

     Укажите желаемый процент расходов на продвижение от дохода, который оно принесло. На указанную величину влияют выбранные [корректировки](ru/impressions/bids-adjustment).
   - **Средняя цена конверсии**

     Стратегия позволяет получить максимальное количество конверсий по выбранным целям по указанной средней цене.

     Средняя цена конверсии рассчитывается как средняя стоимость целевого визита за неделю: отношение недельного расхода к количеству целевых визитов.

     [Рассчитайте цену самостоятельно](ru/strategies/average-cpa#tips) или воспользуйтесь рекомендацией системы. Рекомендуемая цена вычисляется как средняя цена конверсии за неделю по выбранной цели и модели атрибуции. Если в настраиваемой кампании конверсий по цели не было, в расчете участвует средняя цена конверсии по этой цели в других кампаниях.

     В течение недели возможно отклонение средней цены конверсии как в большую, так и в меньшую сторону. В первую неделю после подключения стратегия обучается и средняя цена конверсии может отличаться от заданной.

     На указанную среднюю цену конверсии также влияют выбранные [корректировки](ru/impressions/bids-adjustment).

     Минимальное значение для средней цены конверсии — 0,9 рубля (*значения для других валют*).
   - **Бюджет**

     Стратегия позволяет получить максимальное количество конверсий за указанный недельный бюджет.
2. Добавьте [счетчик Метрики](ru/strategies/average-cpa#conditions).

   Если у вас нет доступа к счетчику Метрики, то можно запросить доступ к целям, которые настроены на нем. Для этого укажите счетчик и нажмите **Запросить доступ**. Далее обратитесь к владельцу счетчика для включения [опции](ru/statistics/metrika#access-to-goals) **Разрешить в рекламных кампаниях оптимизацию по целям**.
3. Выберите целевые действия и задайте их цену или ценность.

   Цель — это действие, к которому вы хотите привести пользователя с помощью продвижения. Можно выбрать получение визитов по целям, заданным в Метрике. Укажите их цену или ценность вручную или выберите ценность — «*Динамическая*» (рекомендуем для eCommerce-целей).

   Определиться с целями для оптимизации поможет цветная точка рядом с ними:

   - Зеленая — по цели достаточно целевых визитов по всем источникам трафика. Рекомендуем использовать эту цель.
   - Серая — по цели мало целевых визитов по всем источникам трафика. Есть риск, что цель будет плохо достигаться и кампания будет работать неэффективно. Выбирайте эту цель, если уверены — целевых визитов достаточно для работы стратегии.

   В стратегии можно указать не более 30 целевых действий.
4. Если необходимо, в дополнительных настройках выберите модель атрибуции.

   Модель атрибуции — это правило, какой переход на сайт считать источником визита. Модель помогает точнее определить вклад рекламы в развитие вашего бизнеса. При назначении ставок система использует статистику Яндекс Метрики по выбранной модели атрибуции и стремится достигать заданного вами результата.

   Выбор модели зависит от типа бизнеса, способа привлечения трафика и ваших задач. По умолчанию установлена автоматическая модель атрибуции. При необходимости вы можете ее изменить.

   Подробнее о [моделях атрибуции](ru/statistics/attribution-model).

Внимание

Проанализируйте работу кампании и укажите адекватные значения параметров. Недостижимое значение средней цены конверсии, недостаточный бюджет или низкая граница максимальной цены клика могут негативно отразиться на работе стратегии. Система в таком случае не сможет поднимать ставки по эффективным ключевым фразам, и целевых визитов будет значительно меньше.

*Целевой визит* — сессия посетителя на сайте, в рамках которой произошло достижение цели.

*Цель* — это желаемое действие, выполненное посетителем на рекламируемом сайте. Например, целью может быть регистрация на сайте, заполнение формы заказа, посещение заданного количества страниц.

Отслеживать достижение целей позволяет [Яндекс Метрика](ru/statistics/metrika).

Достижение одной цели может приносить разный доход, например покупка носков — 50 руб., а платья — 2000 руб. Динамическая ценность передается в Директ для целей электронной коммерции.
