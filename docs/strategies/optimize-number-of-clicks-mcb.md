# Оптимизация кликов для баннера на поиске

- [Принцип работы](ru/strategies/optimize-number-of-clicks-mcb#how-it-works)
- [Настройки стратегии](ru/strategies/optimize-number-of-clicks-mcb#settings)

Стратегия позволяет получить максимальное количество кликов за неделю, при этом в среднем клик будет стоить не больше суммы, которую вы назначили. Стратегия подходит тем, кто знает, сколько готов заплатить за посетителя.

Стратегия эффективна для кампаний с количеством кликов за неделю более 100. Если в кампании используются низкочастотные ключевые фразы и количество кликов за неделю менее 100, возможно отклонение средней цены за клик от заданного значения в большую сторону, но не более чем в 2 раза.

*Средняя цена клика* — отношение недельного расхода бюджета к недельному количеству кликов, то есть цена, которая в среднем списывается за переход посетителя по объявлению.

## Принцип работы

После запуска кампании система будет автоматически назначать ставки по объявлениям так, чтобы привлечь как можно больше пользователей. Вы задаете среднюю цену клика для кампании, и средняя цена клика за неделю не будет превышать заданную сумму. При этом цена в течение недели может меняться, быть выше или ниже назначенной средней цены.

## Настройки стратегии

#### Средняя цена клика

Отношение недельного расхода к недельному количеству кликов, то есть цена, которая в среднем списывается за переход посетителя по объявлению.

Проанализируйте работу кампании и укажите адекватное значение средней цены клика. Недостижимое значение не позволит системе поднимать ставки по эффективным фильтрам, и количество кликов значительно снизится.

В течение дня возможно отклонение средней цены как в большую, так и в меньшую сторону.

Минимальное значение средней цены клика — 3 руб. (*значения для других валют*).

#### Недельный бюджет

Минимальный размер бюджета — 300 руб. (*значения для других валют*).

#### Максимальная ставка

При необходимости вы можете ограничить максимальную ставку.

#### Модель атрибуции

Модель атрибуции — это правило, какой переход на сайт считать источником визита. Модель помогает точнее определить вклад рекламы в развитие вашего бизнеса. При назначении ставок система использует статистику Яндекс Метрики по выбранной модели атрибуции и стремится достигать заданного вами результата.

Выбор модели зависит от типа бизнеса, способа привлечения трафика и ваших задач. По умолчанию установлена автоматическая модель атрибуции. При необходимости вы можете ее изменить.

Подробнее о [моделях атрибуции](ru/statistics/attribution-model).

##### **Целевые действия и их ценность**

Укажите [целевые действия](ru/strategies/priority-goals#priority-goals), чтобы система прицельно работала для решения ваших задач. По умолчанию выбраны *вовлеченные сессии*.

Если у текущего логина нет доступа к нужным счетчикам, вы можете запросить доступ к целям Яндекс Метрики на чужом счетчике и использовать их для настройки.

Количество целевых действий не ограничено.
