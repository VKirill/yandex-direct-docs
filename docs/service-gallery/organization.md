# Продвижение организации в галерее услуг

- [Как настроить в Мастере кампаний](ru/service-gallery/organization#kak-nastroit-v-mastere-kampanij)
- [Как настроить в Единой перфоманс-кампании](ru/service-gallery/organization#kak-nastroit-v-edinoj-perfomans-kampanii)

Если ваша организация имеет рубрики, связанные с оказанием услуг, она может появиться в [галерее услуг](ru/service-gallery/about).

## Как настроить в Мастере кампаний

Запустите продвижение в [Мастере кампаний](ru/campaign-master/about), учитывая следующие особенности.

- Укажите ссылку на сайт или соцсеть. Ваша компания автоматически появится в блоке с опцией **Продвижение организации из Яндекс Бизнеса**. Убедитесь, что опция включена. Если организация не нашлась, добавьте ее вручную.
- Для эффективного продвижения в галерее услуг выберите цели счетчика для профиля организации. Этот счетчик создается автоматически для карточки организации и фиксирует такие действия пользователя в карточке, как клик по кнопке «Позвонить», переход в мессенджер и другие.

## Как настроить в Единой перфоманс-кампании

Запустите продвижение в [Единой перфоманс-кампании](ru/unified-performance-campaign/create-campaign), учитывая следующие особенности.

- Укажите ссылку на сайт или соцсеть. Ваша компания автоматически появится в блоке **Организация из Яндекс Бизнеса**. Если организация не нашлась, добавьте ее вручную.
- Убедитесь, что в блоке **Места показа** включено место показа **Список организаций и галерея услуг**.
- Для эффективного продвижения в галерее услуг добавьте цели счетчика для профиля организации. Этот счетчик создается автоматически для карточки организации и фиксирует такие действия пользователя в карточке, как клик по кнопке «Позвонить», переход в мессенджер и другие.
