# Как оценить результат медийной кампании на странице ya.ru и новой вкладке Браузера

- [Какие показатели отслеживать](../products-cpm-campaign-frontpage/statistics.md#parameters)
- [Как сформировать отчет](../products-cpm-campaign-frontpage/statistics.md#report-wizard)
- [Как принимать решения на основе статистики](../products-cpm-campaign-frontpage/statistics.md#conclusion)

Отчеты о результатах рекламной кампании вы сможете посмотреть после того, как начнутся показы объявлений. Регулярно изучайте статистику, чтобы оценить эффективность кампании.

## Какие показатели отслеживать

Для анализа эффективности медийного баннера важно оценить полученный охват и привлекательность баннера для пользователей.

- **Показы** — количество видимых показов рекламы.
- **Клики** — количество кликов по баннеру.
- **Охват** — количество уникальных пользователей, просмотревших рекламу. При подсчете охвата учитывается история показов на разных устройствах пользователя (смартфонах, планшетах и компьютерах).

## Как сформировать отчет

Перейдите по ссылке **Посмотреть статистику** на странице медийной кампании. В [Мастере отчетов](../statistics/report-wizard.md) постройте отчет по следующим параметрам:

|  |  |
| --- | --- |
| **Параметр** | **Настройка** |
| Группировка | За выбранный период |
| Атрибуция | Последний показ из Яндекс Директа кросс-девайс |
| Столбцы | Выберите:   - Показы - Клики - Охват - Конверсии - Расход всего |

В отчете вы увидите:

- Какой медийный охват принесла вам реклама.
- В каком количестве конверсий участвовала медийная реклама. [Подробнее](../statistics/attribution-model.md#brand-statistics).
- Как часто пользователи взаимодействовали с вашим баннером.

Примечание

Некоторые параметры (например, охват) система рассчитывает на выборке пользователей с помощью методов математической статистики. Это позволяет быстро получить значение параметра на большом объеме данных. Поэтому значение в статистике может немного отличаться от фактического.

[Посмотреть статистику в Директе](https://direct.yandex.ru)

## Как принимать решения на основе статистики

**Увеличить количество показов баннера**

Если кампания выигрывает слишком мало аукционов за показ рекламы, рекомендуем увеличить ставку за 1000 показов (в случае ручной стратегии) или повысить ограничение средней цены за 1000 показов (в случае автоматической стратегии). Сверьтесь с [таблицей](../products-cpm-campaign-frontpage/min-bid.md), корректно ли установлена ставка или средняя цена для разных регионов показа.

**Сравнить баннеры между собой**

В отличие от performance-кампаний, баннеры в одной группе показываются равномерно. Сравнить их можно по количеству кликов — они показывают, насколько баннер заинтересовал пользователей. Добавьте в отчет срез **№ объявления/баннера**, столбцы **Клики** и **CTR (%)** и посмотрите, как отличаются показатели по каждому баннеру.
