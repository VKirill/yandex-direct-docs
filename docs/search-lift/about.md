# Как проводится Search Lift

В основе Search Lift — анализ поисковой активности реальной аудитории кампании. Для этого оценивается реальная аудитория кампании.

Из целевой аудитории кампании отбирается тестовая (90%) и контрольная (10%) группа. Тестовая группа включает пользователей, которые видели объявления. Контрольная группа состоит из пользователей той же целевой аудитории, которым объявления намеренно не демонстрируются. Для создания контрольной группы используется [виртуальный аукцион](../research.md#virtual-auction).
