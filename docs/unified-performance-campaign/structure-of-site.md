# Требования к сайту

Убедитесь, что выполнены все перечисленные ниже условия:

- на сайте есть хотя бы одна страница со списком товарных предложений;
- контент с товарами статичный, а не подгружается на страницу автоматически (например, скриптом). Товары должны присутствовать в HTML-коде страницы ;
- на сайте используется строгая микроразметка — [Каталоги](https://yandex.ru/support/webmaster/supported-schemas/catalogs.html) или [Товары](https://yandex.ru/support/webmaster/supported-schemas/strict-microdata-offers.html);
- у каждого товара в списке товарных предложений есть название и ссылка на отдельную страницу с подробным описанием;
- сайт открыт для индексации в `robots.txt`. Робот обходит сайты с оптимальной скоростью, чтобы избежать лишней нагрузки на сервер. Если данные на сайте меняются чаще, чем проходит индексация, скорость обхода можно настроить вручную. Подробнее в [Справке Вебмастера](https://yandex.ru/support/webmaster/service/crawl-rate.html).

  User-agent роботов, которые индексируют сайт:

  - `Mozilla/5.0 (compatible; YandexBot/3.0; +http://yandex.com/bots)`
  - `Mozilla/5.0 (compatible; YaDirectFetcher/1.0; +http://yandex.com/bots)`

Внимание

Специалисты отдела клиентского сервиса могут вас проконсультировать только по кампаниям того логина, с которого вы обращаетесь. Логин можно увидеть, если открыть [ya.ru](http://ya.ru/) на соседней вкладке браузера. Специалист получит доступ к вашим данным только при обработке обращения.
