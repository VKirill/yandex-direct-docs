# Статистика показов

Получить подробную статистику по кампании и оценить эффективность различных типов объявлений можно с помощью [Мастера отчетов](../statistics/report-wizard.md) и отчета [Поисковые запросы](../statistics/search-queries.md).

> Например, для анализа статистики по товарам и каталогам в [Мастере отчетов](../statistics/report-wizard.md) выберите срез «Тип объявления». Добавьте срезы «Название товара/каталога», «Заголовок», «Текст», «Ссылка», «Категория товара», «Производитель товара». Они позволяют изучить показатели конкретных товаров или каталогов с помощью привычных метрик: показы, клики, конверсии, доход и т. д. Эта информация пригодится для оптимизации кампании. Например, вы сможете выявить позиции, которые приносят больше всего конверсий или дохода и сопоставить эти данные с ДРР.
>
> «Категория товара» и «Производитель товара» отображаются только для товаров. Для каталогов этой информации в отчете не будет. Категория и производитель указываются в отчете на основе информации из фида или микроразметки сайта (если генерация в кампании настроена по сайту).
>
> Если использовать эти же срезы в отчете Поисковые запросы, вы увидите, по каким запросам показывались конкретные товары или каталоги и найдете идеи для доработки фида.

Подробнее о [статистике кампании](../statistics/campaign-report.md).
