# Показы на площадках с контентом для взрослых

Для текстово-графических объявлений в Единой перфоманс-кампании с показами в РСЯ получите дополнительный охват за счет площадок, содержащих контент для взрослых. Это повысит эффективность рекламы в таких тематиках как, например, развлечения, новости, музыка, мероприятия и других.

Объявления взрослой тематики будут всегда показаны только рядом с контентом 18+.

Для объявлений остальных тематик в дополнительных настройках группы в блоке **Тип контента площадок** выберите, как показывать объявления. По умолчанию они не показываются рядом с контентом для взрослых.

- **Показывать рядом с любым контентом** — чтобы размещаться на всех площадках.
- **Показывать только рядом с контентом для взрослых** — если вы хотите выделить показы на площадках со взрослым контентом для раздельной оценки статистики или использовать в объявлениях специальные креативы.

Площадки с контентом для взрослых представлены сервисами [Яндекс Картинки](http://images.yandex.ru/) и [Яндекс Видео](http://video.yandex.ru/). Если вы решили показывать на них объявления, проверьте, что они не добавлены в список площадок для [запрета показов](../campaigns/blocked-sites.md) в настройках кампании.
